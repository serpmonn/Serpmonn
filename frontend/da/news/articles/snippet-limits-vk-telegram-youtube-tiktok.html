Tegnbegrænsninger og snippet-afkortning (VK / Telegram / YouTube / TikTok)

15. september 2025

1) Oversigtstabel over begrænsninger og anbefalinger

Bemærk: platforme ændrer ofte UI og afkortning. Vi giver testede anbefalede længder og kendte hårde grænser.

Platform Titel (anbef./grænse) Beskrivelse/tekst (anbef./grænse) Forhåndsvisning/afkortning
VK ≤ 60–70 / — ≤ 140–160 / op til 4096 tegn i indlæg Link-snippet afkorter titel ~70, beskrivelse ~140+
Telegram ≤ 70 / — (hentet fra OG) ≤ 120–160 / 4096 i besked Forhåndsvisning fra OG: titel ~70, beskrivelse ~120–140
YouTube ≤ 60 / grænse 100 ≤ 150–200 (første linjer) / grænse 5000 Above-the-fold viser ~2–3 linjer
TikTok ≤ 60–80 / — ≤ 80–150 (synlig del) / grænse op til ~2200 Billedtekst afkortes, 'Mere' efter ~80–150

2) VK: før/efter

FØR

"En utrolig detaljeret og meget lang titel om vores nye funktion, der uundgåeligt vil blive afkortet i alle forhåndsvisninger og reducere CTR..."

EFTER

"Ny funktion: hurtigere og enklere" — kort og klart. Undertekst/beskrivelse: "Vi har samlet nøglefunktioner og brugseksempler" (≤ 140).

3) Telegram: før/efter

FØR

"[Ny artikel] Komplet guide til UTM-tags for alle marketingkanaler, med lister, eksempler, tjekliste og indholdsfortegnelse"

EFTER

"UTM: komplet guide" — "GA4, skabeloner til Google/Meta/VK, almindelige fejl". Titel ≤ 70, beskrivelse ≤ 120–140.

4) YouTube: før/efter

FØR

"Hvordan vi byggede SEO på 30 dage: fuld gennemgang af alle trin fra audit til implementering, analyse og rapportering [2025]" (105 tegn)

EFTER

"SEO på 30 dage: trin-for-trin plan" — "Audit, implementering, rapportering. Fejl og life hacks (2025)". Titel ≤ 60, første 150–200 tegn af beskrivelse — nøglefordele og CTA.

5) TikTok: før/efter

FØR

"Se vores nye lange forklaring med masser af detaljer og links — sørg for at læse til slutningen!"

EFTER

"3 UTM-fejl, der æder trafik" — "Tjek source/medium, kodning og omdirigeringer. Komplet guide — via link".

6) Tips til forhåndsvisninger og metadata

  • Sæt OG-meta (title/description/image) for korrekte forhåndsvisninger i VK/Telegram.
  • Hold title ≤ 60–70, description ≤ 140–160 for universel synlighed.
  • Placer nøglefordele i de første 1–2 linjer af beskrivelsen.
  • Tjek forhåndsvisninger før udgivelse (via kladder/forhåndsvisningsbots/deling).
