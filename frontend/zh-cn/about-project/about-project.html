Serpmonn.ru 是什么？

Serpmonn 是一个综合性平台，专注于实用工具：身份验证和个人资料、RSS 新闻、带有排行榜的浏览器游戏和 Telegram 机器人（游戏、验证、表情包）。特别关注安全性：核心服务中添加了 CSRF 端点和速率限制。Serpmonn.ru 的目标是将这些功能结合在一个方便的界面中，用于日常任务。

项目公开开发：移动端和桌面端的用户体验逐渐改善，工具不断添加，可访问性得到扩展（大字体、高对比度、更少动画）。计划包括更多 HTML5 游戏、改进的搜索、集成和完整的个人资料系统。

项目功能

  • 身份验证和个人资料 — 安全授权（PASETO），个人资料管理。
  • 信息分析工具 — 7个数据收集和处理服务，RSS 新闻。
  • 平台游戏 — 19个带有排行榜的浏览器和 PC 游戏。
  • Telegram 机器人 — 游戏机器人、验证机器人和表情包机器人。
  • PWA — 支持渐进式 Web 应用。
  • 安全 — 核心服务中的 CSRF 端点和速率限制。

谁可以受益

  • 用户 — 在一个地方方便地访问新闻、游戏和服务。
  • 开发者 — 具有清晰结构的开放仓库，可以贡献。
  • 机器人和迷你服务作者 — 用于集成和测试想法的平台。

已经运行的功能

  • 身份验证和个人资料系统。
  • 来自 RSS 源的新闻（9篇文章）。

安全

项目正在逐步加强：使用 PASETO 令牌，实施 CSRF 端点，添加速率限制，更新传递依赖项。执行静态分析（CodeQL）和定期依赖项更新。

技术

  • 后端：Node.js, Express, MySQL, PASETO, PM2.
  • 前端：HTML, CSS, JavaScript, PWA.
  • 基础设施：Nginx, CI/CD, CodeQL.
