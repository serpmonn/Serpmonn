پورت يۆللىتىش: تولۇق يېتەكچى 2025

29-ئۆكتەبىر 2025 • 15 مىنۇت ئوقۇش

پورت يۆللىتىش ئۆي تورىنى تەڭشەشتە يېڭى باشلىغۇچىلار ئۈچۈن ئەڭ كۆپ تەلەپ قىلىنىدىغان ۋە ئەڭ مۇرەككەپ مەزمۇنلارنىڭ بىرى. بۇ يېتەكچىدە، بىز ھەر قانداق مەقسەت ئۈچۈن پورت يۆللىتىشنى قانداق توغرا تەڭشەشنى تولۇق چۈشەندۈرىمىز: ئويۇن مۇلازىمېتىرلىرىدىن كۆزىتىش سىستېمىلىرىغىچە.

🔍 پورت يۆللىتىش نېمە؟

پورت يۆللىتىش تور تەلەپلىرىنى روتېرنىڭ سىرتقى پورتىدىن سىزنىڭ يەرلىك تورىڭىزدىكى قورالنىڭ مەلۇم بىر پورتىغا قايتا يۆنەلتۈشكە يول قويىدىغان تېخنولوگىيە.

⚠️ چۈشىنىش مۇھىم

پورت يۆللىتىش قورالىڭىزغا ئىنتېرنېتتىن زىيارەت قىلىشنى ئاچىدۇ. ھەمىشە بىخەتەرلىك تەدبىرلىرىگە ئەمەل قىلىڭ ۋە زۆرۈر ئەمەس پورتلارنى ئېچماڭ!

🔧 تەڭشەشكە تەييارلىق

  1. قورالنىڭ مۇقىم IP ئادرېسىنى بەلگىلەڭ

    قورالىڭىز (كومپيۇتېر، مۇلازىمېتىر، كامېرا) روتېر ھەرىكەتنى قەيەرگە قايتا يۆنەلتەيدىغانلىقىنى بىلىشى ئۈچۈن داۋاملىق يەرلىك IP ئادرېسىگە موھتاج.

    Windows'ta IP ئادرېسىنى قانداق تاپقىلى بولىدۇ:

    ipconfig

    تور باغلىنىشىڭىز ئۈچۈن "IPv4 Address" قۇرىنى ئىزدەڭ

    Linux/macOS'ta IP ئادرېسىنى قانداق تاپقىلى بولىدۇ:

    ifconfig

    ياكى ئىشلىتىڭ: ip addr show

  2. مۇقىم IP تەڭشەڭ

    IP ئادرېسىنىڭ ئۆزگىرىشىنىڭ ئالدىنى ئېلىش ئۈچۈن، ئۇنى مۇقىم تەڭشەڭ:

    • روتېردا: DHCP زاپىسى ئارقىلىق مۇقىم IP بەلگىلەڭ
    • قورالدا: تور تەڭشەكلىرىدە قولدا مۇقىم IP تەڭشەڭ

  3. سىرتقى IP ئادرېسىڭىزنى بىلىڭ

    بۇ باشقا ئىشلەتكۈچىلەر ئىنتېرنېتتا كۆرىدىغان ئادرېس:

    سىرتقى IP نى بىلىش يوللىرى:

    # بۇيرۇق قۇرى ئارقىلىق
curl ifconfig.me

# ياكى تور بەتلەرنى زىيارەت قىلىڭ:
# - whatismyipaddress.com
# - 2ip.ru
# - myip.com

  4. زۆرۈر پورتلارنى بەلگىلەڭ

    ئىشلىتىشىڭىز ياكى مۇلازىمىتىڭىز قايسى پورتلارنى ئىشلىتىدىغانلىقىنى بىلىڭ. بۇ ئۇچۇر ئادەتتە ھۆججەتلەردە بولىدۇ.

🚀 قەدەممۇ-قەدەم تەڭشەش كۆرسەتمىسى

  1. روتېر كونترول پانىلىغا كىرىڭ

    تور كۆرگۈچىڭىزنى ئېچىڭ ۋە روتېرنىڭ IP ئادرېسىنى كىرگۈزۈڭ. ئادەتتە بۇلار:

    • 192.168.1.1 (TP-Link, D-Link)
    • 192.168.0.1 (Netgear, بەزى TP-Link)
    • 192.168.31.1 (Xiaomi)
    • 192.168.10.1 (Huawei)

    ئىشلەتكۈچى ئىسمى / ئىم قىممىتى ئادەتتە روتېردىكى ئەتكەزدە كۆرسىتىلگەن.

  2. پورت يۆللىتىش بۆلۈمىنى تېپىڭ

    بۆلۈمنىڭ ئوخشىمىغان ئىسىملىرى بولۇشى مۇمكىن:

    • Port Forwarding
    • Virtual Servers
    • پورت يۆللىتىش
    • NAT
    • ئوتتۇرىغا قويۇلغان ئىشەنچلىك دېۋار

  3. پورت يۆللىتىش پارامېتىرلىرىنى تولدۇرۇڭ

    سىز تۆۋەندىكىلەرنى بەلگىلىشىڭىز كېرەك:

    • مۇلازىمەت ئىسمى: ھەر قانداق ئىسىم (مەسىلەن "ماینكرافت مۇلازىمېتىرى")
    • سىرتقى پورت: روتېردا ئېچىلىدىغان پورت
    • ئىچكى پورت: قورالىڭىزدىكى پورت
    • ئىچكى IP: قورالىڭىزنىڭ مۇقىم IP
    • پروتوكول: TCP، UDP ياكى ئىككىسى

  4. تەڭشەكلەرنى ساقلاڭ

    بارلىق سۆز بۆلەكلىرىنى تولدۇرغاندىن كېيىن، "ساقلا" ياكى "قوللى" دېگەننى بېسىڭ. روتېر قايتا قوزغىلىشى مۇمكىن.

📡 ئاممىباب روتېرلار ئۈچۈن كۆرسەتمە

🔄 TP-Link

  1. كونترول پانىلىغا كىرىڭ (192.168.1.1)
  2. بارىڭ: ئىلغار → NAT يۆنەلتىش → ۋىرتىئال مۇلازىمېتىرلار
  3. "Add New" نى چېكىڭ
  4. تولدۇرۇڭ:
    • Service Type: تىزىمدىن تاللاڭ ياكى ئۆزىڭىزنى كىرگۈزۈڭ
    • External Port: سىرتقى زىيارەت قىلىش ئۈچۈن پورت
    • Internal Port: قورالدا پورت
    • IP Address: قورالنىڭ يەرلىك IP
    • Protocol: TCP/UDP/BOTH
  5. ساقلاڭ ۋە روتېرنى قايتا قوزغىتىڭ

🛡️ ASUS

  1. پانىلغا كىرىڭ (192.168.1.1)
  2. بارىڭ: ئىلغار تەڭشەك → WAN → ۋىرتىئال مۇلازىمېتىر / پورت يۆللىتىش
  3. "Enable Port Forwarding" نى قوزغىتىڭ
  4. سۆز بۆلەكلىرىنى تولدۇرۇڭ:
    • Service Name: مۇلازىمەت ئىسمى
    • Port Range: پورت دائىرىسى (مەسىلەن 25565-25565)
    • Local IP: قورالىڭىزنىڭ IP
    • Local Port: قورالىڭىزنىڭ پورتى
    • Protocol: TCP/UDP/BOTH
  5. "Add" كېيىن "Apply" نى چېكىڭ

🔗 D-Link

  1. پانىلغا كىرىڭ (192.168.0.1)
  2. بارىڭ: ئىلغار → پورت يۆللىتىش
  3. "Add" نى چېكىڭ
  4. تولدۇرۇڭ:
    • Rule Name: قائىدە ئىسمى
    • External Port: سىرتقى پورت
    • Internal Port: ئىچكى پورت
    • Internal IP: قورالنىڭ IP
    • Schedule: ھەمىشە
    • Inbound Filter: Allow All
  5. تەڭشەكلەرنى ساقلاڭ

📶 Zyxel Keenetic

  1. پانىلغا كىرىڭ (my.keenetic.net ياكى 192.168.1.1)
  2. بارىڭ: بىخەتەرلىك → پورت يۆللىتىش
  3. "قائىدە قوشۇش" نى چېكىڭ
  4. تولدۇرۇڭ:
    • چۈشەندۈرۈش: مۇلازىمەت ئىسمى
    • ئارىيىز: ئىنتېرنېت
    • پروتوكول: TCP/UDP
    • سىرتقى پورت: سىرتقى زىيارەت قىلىش ئۈچۈن پورت
    • نىشان IP: قورالنىڭ يەرلىك IP
    • ئىچكى پورت: قورالدا پورت
  5. ئۆزگەرتىشلەرنى ساقلاڭ

✅ پورت يۆللىتىش ئىشىنى تەكشۈرۈش

  1. تور مۇلازىمەتلىرىنى ئىشلىتىڭ

    ئېچىلغان پورتلارنى تەكشۈرۈش ئۈچۈن تور بەتلەر:

    • portchecker.co
    • canyouseeme.org

  2. بۇيرۇق قۇرى ئارقىلىق تەكشۈرۈش

    باشقا كومپيۇتېردىن تەكشۈرۈڭ:

    # telnet ياكى nc (netcat) ئىشلىتىڭ
telnet سىزنىڭ-سىرتقى-IP پورت
# ياكى nmap ئىشلىتىپ
nmap -p پورت سىزنىڭ-سىرتقى-IP

  3. مۇلازىمەت سىناق

    يەرلىك تورىڭىزنىڭ سىرتىدىكى باشقا قورال ئارقىلىق (مەسىلەن تاشقىن ئىنتېرنېت ئارقىلىق) مۇلازىمىتىڭىزگە ئۇلىنىشقا تىرىشىڭ.

✅ مۇۋەپپەقىيەتلىك تەڭشەش ئالامەتلىرى

  • پورت توردىكى تەكشۈرگۈچىلەردە "ئېچىق" دەپ كۆرۈنىدۇ
  • سىز سىرتقى تەرەپتىن مۇلازىمەتكە ئۇلىنالايسىز
  • مۇلازىمەت توختىماستىن مۇقىم ئىشلەيدۇ

🔧 ئادەتتىكى مەسىلىلەرنى ھەل قىلىش

🚫 پورت تېخىمۇ تاقالغان

  • ئوتتۇرىغا قويۇلغان ئىشەنچلىك دېۋارنى تەكشۈرۈڭ: Windows ئوتتۇرىغا قويۇلغان ئىشەنچلىك دېۋار ياكى باشقا ئوتتۇرىغا قويۇلغان ئىشەنچلىك دېۋاردا ئىستىسنا قوشۇڭ
  • روتېرنى قايتا قوزغىتىڭ: بەزىدە تەڭشەكلەر پەقەت قايتا قوزغىتىلغاندىن كېيىنلا قوللىنىلىدۇ
  • IP ئادرېسىنى تەكشۈرۈڭ: قورالنىڭ توغرا مۇقىم IP گە ئىگە ئىكەنلىكىگە ئىشەنچ ھاسىل قىلىڭ
  • UPnP نى چەكلەڭ: بەزىدە UPnP قولدا پورت تەڭشىكى بىلەن توقۇنۇشىدۇ

⚠ى تەمىنلىگۈچىنىڭ ھەرىكەتچان IP

ئەگەر سىزنىڭ ھەرىكەتچان IP ىڭىز بولسا، ئۇ ئۆزگىرىشى مۇمكىن. ھەل قىلىش ئۇسۇللىرى:

  • DDNS ئىشلىتىڭ: No-IP, DuckDNS قاتارلىق مۇلازىمەتلەر
  • IP نى تەكشۈرۈڭ: نۆۋەتتىكى IP ھەققىدىكى ئۇچۇرلارنى مۇنتىزىم يېڭىلاڭ
  • مۇقىم IP: تەمىنلىگۈچىدىن مۇقىم IP تەلەپ قىلىڭ (پۇل تۆلەنگەن)

🔒 قوش NAT

تورىڭىزدا بىر نەچچە روتېر بولغاندا مەسىلە يۈز بېرىدۇ:

  • كۆۋرۈك ھالىتى: روتېرلارنىڭ بىرىنى كۆۋرۈك ھالىتىگە تەڭشەڭ
  • DMZ: ئىككىنچى روتېر ئۈچۈن ئاساسىي روتېرda DMZ تەڭشەڭ
  • ئىككىسىدە يۆنەلتىش: ئىككى روتېردا يۆنەلتىشنى تەڭشەڭ

🛡ى بىخەتەرلىك تەدبىرلىرى

🔐 مۇھىم قائىدىلەر

  1. پەقەت زۆرۈر پورتلارنى ئېچىڭ - ھەر بىر ئېچىلغان پورت ھۇجۇم يۈزىنى ئاشۇرىدۇ
  2. سىرتقى تەرەپتىن زىيارەت قىلغىلى بولىدىغان بارلىق مۇلازىمەتلەردە مۇرەككەپ ئىم ئىشلىتىڭ
  3. يۇمشاق دېتالنى مۇنتىزىم يېڭىلاڭ - ئاجىزلىقلارنى يوقىتىڭ
  4. روتېر خاتىرىلىرىنى ئىشلىتىپ باغلىنىشلارنى كۆزىتىڭ
  5. پورت يۆللىتىشكە بىخەتەرلىك مۇبا دەل سۈپىتىدە VPN نى ئويلىشىڭ

🎯 ئەڭ ياخشى ئەمەلىيەت

  • سىكانېرلارنىڭ دىققىتىنى ئازايتىش ئۈچۈن ستاندارت ئەمەس پورتلارنى ئىشلىتىڭ
  • جەدۋەل بويىچە پورتلارنىڭ ئاپتوماتىك چەكلەشنى تەڭشەڭ
  • گۇمانلىق ھەرىكەت ئۈچۈن خاتىرىلەرنى مۇنتىزىم تەكشۈرۈڭ
  • گۇمانلىق IP لارنى چەكلەش ئۈچۈن fail2ban ياكى ئوخشاشنى ئىشلىتىڭ
