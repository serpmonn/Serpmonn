ކުރީގެ ފޮތުގެ މައުލޫމާތު

  • ތިޔަބޭފުޅުން ކިޔަންނަނީ ކޮންބައެއް
  • ލޯރސް ޓޫލްސް
  • ސްޓެޕް ބައި ސްޓެޕް ގައިޑް
  • އާދައިގެ ޓޫލްސް
  • އެންމެ ފަހުގެ ސަވާލް
  • ފައިދާ މަސްދަރުތައް
ޑިވެލޮޕްމަންޓް

ކޮންފަދައަށް ތިޔަބޭފުޅުންގެ ފުރަތަމަ ވެބްސައިޓެއް ނުސްކަން އުފެއްދުން

އޮގަސްޓް 18، 2024 • 12 މިނިޓް ކިޔުން

ވެބްސައިޓެއް ނުސްކަން އުފެއްދުން ފުރަތަމަ މިންގަނޑަކަށް ބޮޑު މަސައްކަތެއް ފެނުނުގޮތަކަށް ފެންނަނީ، އެހެނެއް އެއީ އެންމެ ސަބަބުން އެންމެ މިންގަނޑުގެ މަސައްކަތެއް ކަމުގައިވާ ހަމައެކަނި މަސައްކަތެއް ކަމުގައި ވެއެވެ. މި ފޮތުގައި، އަހަރެން އިންތިޒާމުކުރެވޭ ވެބްސައިޓެއް ނުސްކަން އުފެއްދުންގެ މަތީގައި އެންމެ ބޮޑު މަސައްކަތް ކުރަމުންދެން އެބަހުރިހުރި ފަރާތްތައް ދައްކައިދޭނެއެވެ.

ތިޔަބޭފުޅުން ކިޔަންނަނީ ކޮންބައެއް

  • HTML، CSS އަދި JavaScript ގެ ބަސްފޮތް
  • ކޮންފަދައަށް ހޯސްޓިންގ އަދި ޑޮމެއިން އިޚްތިޔާރު ކުރެވޭނެއެއް
  • އާދައިގެ ޑިވެލޮޕްމަންޓް ޓޫލްސް
  • އިންޓަރނެޓްގައި ވެބްސައިޓް ޕަބްލިޝް ކުރުން

ފައިލް މެނެޖަރ

ޕްރޮޖެކްޓް އޮގަނައިޒޭޝަން އަށް

ސްޓެޕް ބައި ސްޓެޕް ގައިޑް

ޕްލެންނިންގ

މަކްސަދު އަދި ސައިޓް ސްޓްރަކްޗަރ ޑިފައިން ކުރުން

ޑިވެލޮޕްމަންޓް

HTML، CSS އަދި JavaScript އުފެއްދުން

ޕަބްލިޝިންގ

އިންޓަރނެޓްގައި ވެބްސައިޓް ޕަބްލިޝް ކުރުން

  1. ޕްލެންނިންގ އަދި ޑައިޒައިން

    ވެބްސައިޓްގެ މަކްސަދު، ޓާގެޓް އެޑިއުންސް އަދި މެއިން ފީޗަރސް ޑިފައިން ކުރުން. ކައުންޓަރ އޮން ޕޭޕަރ ނުވަތަ ގްރާފިކް އެޑިޓަރގައި ސިމްޕަލް ލޭއައުޓް އެއް ނުސްކަން އުފެއްދުން.

  2. HTML ސްޓްރަކްޗަރ އުފެއްދުން

    ބެސިކް HTML ސްޓްރަކްޗަރއާ ފަހަރު ފެށުން. HTML އަކީ ތިޔަބޭފުޅުންގެ ވެބްސައިޓްގެ ސްކެލެޓަށް އެދެނީ.

    ބެސިކް HTML ޓެމްޕްލެޓް:

    <!DOCTYPE html>
<html lang="dv">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>މަގޭ ފުރަތަމަ ވެބްސައިޓް</title>
</head>
<body>
    <h1>މަރުޙަބާ!</h1>
    <p>މިއީ މަގޭ ފުރަތަމަ ވެބްސައިޓް.</p>
</body>
</html>
    HTML އެޑިޓަރ

  3. CSS އަށްޓަކައި ސްޓައިލިންގ

    CSS އަކީ ވެބްސައިޓްގެ ފެންވަރަށް މަސޫލް ކަމުގައި ވެއެވެ. ސިމްޕަލް ސްޓައިލްސްން ފަހަރު ފެށުން އަދި ގްރޭޑިއުއަލް ވަގުތުން އެތައް ކޮމްޕްލެކްސް ކުރުން.

    ބެސިކް CSS ސްޓައިލްސް:

    body {
    font-family: Arial, sans-serif;
    margin: 0;
    padding: 20px;
    background-color: #f0f0f0;
}
h1 {
    color: #333;
    text-align: center;
}
p {
    line-height: 1.6;
    color: #666;
}
    CSS އެޑިޓަރ

    މަސަލް މަތިންގ

    CSS ސްޓައިލިންގ އަށްޓަކައި މަސަލް ތެކްސްޓް

  4. އިންޓަރެކްޓިވިޓީ އެޑް ކުރުން

    ޖާވާސްކްރިޕްޓް ވެބްސައިޓް އިންޓަރެކްޓިވް ކުރައްވައިދެއެވެ. ސިމްޕަލް ފަންކްޝަންސްން ފަހަރު ފެށުން.

    ސިމްޕަލް JavaScript:

    document.getElementById('myButton').addEventListener('click', function() {
    alert('އައްސަލާމް ދުނިޔެ!');
});

  5. ޓެސްޓިންގ

    ވެބްސައިޓް ވެރިއައިސް ބްރައުޒަރ އަދި ޑިވައިސްގައި އޮޕަން ކުރުން. ހުރިހާ ކަމެއް ރައީގަތް ވަރަށް ކޮށްދޭނެ ކަމަށް ޔަޤީން ކުރުން.

  6. ޕަބްލިޝިންގ

    ހޯސްޓިންގ އިޚްތިޔާރު ކުރުން (މިސާލަކަށް، GitHub Pages، Netlify ނުވަތަ Vercel) އަދި ވެބްސައިޓް ފައިލްސް އަޕްލޯޑް ކުރުން.

ފުރަތަމަ މިންގަނޑުންގެ ޓިޕް

އަލުން ކޮމްޕްލެކްސް ވެބްސައިޓް އެއް ނުސްކަން އުފެއްދުމަށް މަސައްކަތް ނުކުރާށެވެ. ސިމްޕަލް ސިންގަލް ޕޭޖް ވެބްސައިޓްއަށް ފަހަރު ފެށުން އަދި ގްރޭޑިއުއަލް ވަގުތުން ނުވައި ފީޗަރސް އެޑް ކުރުން.

އާދައިގެ ޓޫލްސް

އިތުރު އާންމު ޑިވެލޮޕްމަންޓް އަށް، ހިއްސާ ކުރައްވާ:

  • ފްރެއިމްވާކްސް: React، Vue.js، Angular
  • CSS ފްރެއިމްވާކްސް: Bootstrap، Tailwind CSS
  • ބިލްޑް ޓޫލްސް: Webpack، Vite، Parcel
  • ވާޖަން ކޮންޓްރޯލް ސިސްޓަމްސް: Git

ހަނދުމަކުރެވޭ މައުލޫމާތު

ވެބް ޑިވެލޮޕްމަންޓް އަކީ މައުގައި އަންނަ މަސައްކަތެއް. އާދަ ޓެކްނޮލޮޖީތައް ދަސްކުރުމަށް އަދި އިންޑަސްޓްރީ ޓްރެންޑްސް ފޮލޯ ކުރުމަށް ތައްޔާރު ވެއްޖެ.

އެންމެ ފަހުގެ ސަވާލް

ތިޔަބޭފުޅުންގެ ފުރަތަމަ ވެބްސައިޓް ނުސްކަން އުފެއްދުމަށް ކިތައް ވަގުތު ގެންދެއެވެ؟

ސިމްޕަލް ބިޒްނަސް ކާޑް ވެބްސައިޓްއަށް ބޭނުންކުރާ ވަގުތު އެއްވަނީ މިނިސްތަކެއް ނުވަތަ ދުވަސްތަކެއް. އިތުރު ކޮމްޕްލެކްސް ޕްރޮޖެކްޓްތަކަށް ވަގުތު އެއްވަނީ ހަފްތާ ނުވަތަ މަސްތަކެއް، ކޮމްޕްލެކްސިޓީ އަދި ތިޔަބޭފުޅުން ބޭނުންކުރައްވާ ވަގުތު މިނިވަންކަމުގައި ބަދަލުވެގެންދެއެވެ.

ވެބް ޑިވެލޮޕްމަންޓް އަށް ތިޔަބޭފުޅުން މެތްތު ދަސްކުރުމަށް ބޭނުންވޭތޯ؟

ގިނަ ވެބް ޕްރޮޖެކްޓްތަކަށް އެޑްވާންސް މެތްތު ބޭނުންނުވޭ. ތިޔަބޭފުޅުން ބޭނުންވާ މުހިންމު ސްކިލްސް އަކީ ލޮޖިކަލް ޘިންކިންގ އަދި ޕްރޮބްލަމް ސޮލިއުޝަން.

ކޮން ޕްރޮގްރާމިންގ ލެންގުއޭޖް ފުރަތަމަ ދަސްކުރައްވާ ހެޔޮކަމެއް ހެއްޔެވެ

HTML އަދި CSS އަކީ ވެބްގެ ފަށްޓަށް އެދެނީ އަދި ފުރަތަމަ ދަސްކުރައްވާ ހެޔޮކަމެއް ކަމުގައި ވެއެވެ. ދެން ޖާވާސްކްރިޕްޓްއަށް ވަޑައިގަތުން ވެބްސައިޓްއަށް އިންޓަރެކްޓިވިޓީ އެޑް ކުރައްވައިދެއެވެ.

ފައިދާ މަސްދަރުތައް

  • MDN ވެބް ޑޮކްސް (ދިވެހި) — އެލާ ޑޮކިއުމެންޓޭޝަން
  • W3Schools — އިންޓަރެކްޓިވް ލެސަންސް
  • GitHub — ހޯސްޓިންގ އަދި ކޯޑް ލާރނިންގ އަށް
  • Stack Overflow — ޑިވެލޮޕަރސް އަށް ގުސްޓްޝަންސް އަދި އެންސަރސް

ވެބްސައިޓެއް ނުސްކަން އުފެއްދުން އަކީ އެންމެ ފަހުގެ ހުރިހާ މިންގަނޑުންގެ މަސައްކަތެއް ކަމުގައިވާ އިޖާދީ މަސައްކަތެއް ކަމުގައި ވެއެވެ. ޓްރައި ކުރުމަށް އަދި ތިޔަބޭފުޅުންގެ ކުށްތަކުން ދަސްކުރުމަށް ބިރުނުބަލާށެވެ!
