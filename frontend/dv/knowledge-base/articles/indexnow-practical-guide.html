IndexNow: ޕްރެކްޓިކަލް އިމްޕްލިމެންޓޭޝަން އަދި ޓިޕްސް

15 ސެޕްޓެމްބަރ 2025 • 5 މިނެޓް ކިޔުމު
ކީއްވެ ކިޔާށެވެ: ކޮށްފައިވާ އަޕްޑޭޓްސް ސާރޗު އެންޖަންސްއަށް ކިތަން ވަގުތު އެނގޭނެ، ކޮން ވަގުތު އޮންލީ-ޗޭންޖްޑް ބޭނުންކުރާށެވެ، IndexNow ކިތަން ސައިޓްމެޕް ޕިންގ އަށް ވަކިވަނީ އަދި ކިތަން އާދަ ކުށްތައް އެއްކޮށްލާށެވެ
ތަފްސީލް

  2. ކީ ސެޓަޕް އަދި ބޭސިކް ޕިންގ
  3. އޮންލީ-ޗޭންޖްޑް މޯޑް: ކޮން ވަގުތު އަދި ކިތަން
  4. IndexNow ވާސް ސައިޓްމެޕް ޕިންގ
  5. އާދަ ކުށްތައް
  6. ފްރީކޭންޓްލީ އާސްޓް އަދި ޗެކްލިސްޓް

1) IndexNow ކަމަކީ

IndexNow އަކީ އެކި ޕްރޮޓޮކޮލެކެވެ، އެއް އިންސާނަކީ ސާރޗު އެންޖަންސްއަށް URL ބަދަލުތައް (އުފެއްދުން، އަޕްޑޭޓް، ފޮހެލުން) ކުރިއަށް އަދާކުރައްވާފައިވާ އެއްޗެކެވެ. Bing، Yandex، Seznam އަދި އެހެން ސަރވިސްތައް ދެއްވާފައިވެއެވެ. ފައިދާ އަކީ އެންޑެކްސިންގ ދަތިކަމަށް އަދި ކްރޯލް ނޫސް އިތުރަށް ކަމަށެވެ.

2) ކީ ސެޓަޕް އަދި ބޭސިކް ޕިންގ

  1. އެކި ކީ (GUID) ސެޕްޓް ކުރޭ އަދި YOUR_KEY.txt ފައިލް ވެބްސައިޓްގެ ރޫޓްގައި (/.well-known/ ނުވަތަ ޑޮމެއިން ރޫޓް) ކީ ސްޓްރިންގ އަށް ތައްޔާރު ކުރޭ.
  2. URL އަދި ކީ އަށް އެންޑްޕޮއިންޓަށް ޕިންގ ފޮނުއްވޭ. މިސާލު (އެކި URL):
POST https://api.indexnow.org/indexnow Content-Type: application/json { "host": "www.serpmonn.ru", "key": "YOUR_KEY", "urlList": [ "https://www.serpmonn.ru/frontend/knowledge-base/articles/utm-complete-guide.html" ] }
ބެޗްތަކަށް، އެކި ރިކޯސްޓްގައި 10,000 އިން މައްސަލަ URL ސެޓް ކުރޭ ނުވަތަ ޗޭންޖް ލޮގް ފައިލް URL އެއް ބޭނުންކުރޭ ފައިލް ފީޑް.

3) އޮންލީ-ޗޭންޖްޑް މޯޑް

އިދިޔާއަކީ އެންމެ ފަހުގެ commit/ޑެޕްލޯމަންޓްގައި ހަމަ ބަދަލުވެފައިވާ ފައިލްތަކަށް އޮންލީ ޕިންގ ފޮނުއްވުން. މިއީ ނޫސް އަދި ކޯޓާ ޔޫޒް ކަމަށް އެއްކޮށްދެއްވުން.
