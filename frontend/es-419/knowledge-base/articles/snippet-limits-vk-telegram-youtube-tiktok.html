Límites de caracteres y truncamiento de snippets (VK / Telegram / YouTube / TikTok)

15 de septiembre de 2025
Por qué leer esto: para que los títulos y descripciones no se corten y ofrezcan el máximo CTR. A continuación — tabla resumen de límites y ejemplos visuales 'antes/después' para corregir.

1) Tabla resumen de límites y recomendaciones

Nota: las plataformas cambian con frecuencia la UI y el truncamiento. Ofrecemos longitudes recomendadas probadas y límites duros conocidos.

Plataforma Título (recom./límite) Descripción/texto (recom./límite) Vista previa/truncamiento
VK ≤ 60–70 / — ≤ 140–160 / hasta 4096 caracteres en la publicación El snippet de enlace trunca título ~70, descripción ~140+
Telegram ≤ 70 / — (tomado de OG) ≤ 120–160 / 4096 en el mensaje Vista previa desde OG: título ~70, descripción ~120–140
YouTube ≤ 60 / límite 100 ≤ 150–200 (primeras líneas) / límite 5000 Above-the-fold muestra ~2–3 líneas
TikTok ≤ 60–80 / — ≤ 80–150 (parte visible) / límite hasta ~2200 El pie de foto se trunca, 'Más' después de ~80–150

2) VK: 'antes/después'

ANTES

"Un título increíblemente detallado y muy largo sobre nuestra nueva función que inevitablemente se truncará en todas las vistas previas y reducirá el CTR…"

DESPUÉS

"Nueva función: más rápida y sencilla" — breve y claro. Subtítulo/descripción: "Recopilamos las capacidades clave y ejemplos de uso" (≤ 140).

3) Telegram: 'antes/después'

ANTES

"[Nuevo artículo] Guía completa de etiquetas UTM para todos los canales de marketing, con listas, ejemplos, lista de verificación y tabla de contenidos"

DESPUÉS

"UTM: guía completa" — "GA4, plantillas para Google/Meta/VK, errores comunes". Título ≤ 70, descripción ≤ 120–140.

4) YouTube: 'antes/después'

ANTES

"Cómo construimos SEO en 30 días: desglose completo de todos los pasos desde la auditoría hasta la implementación, análisis e informes [2025]" (105 caracteres)

DESPUÉS

"SEO en 30 días: plan paso a paso" — "Auditoría, implementación, informes. Errores y consejos prácticos (2025)". Título ≤ 60, primeros 150–200 caracteres de la descripción — beneficios clave y CTA.

5) TikTok: 'antes/después'

ANTES

"Mira nuestra nueva explicación larga con muchos detalles y enlaces — ¡asegúrate de leer hasta el final!"

DESPUÉS

"3 errores de UTM que consumen tráfico" — "Verifica source/medium, codificación y redirecciones. Guía completa — mediante enlace".

6) Consejos para vistas previas y metadatos

  • Configura meta OG (title/description/image) para vistas previas correctas en VK/Telegram.
  • Mantén title ≤ 60–70, description ≤ 140–160 para visibilidad universal.
  • Coloca el beneficio clave en las primeras 1–2 líneas de la descripción.
  • Verifica las vistas previas antes de publicar (mediante borradores/bots de vista previa/compartiendo).
