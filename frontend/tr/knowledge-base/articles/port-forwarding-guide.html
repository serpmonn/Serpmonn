Port Yönlendirme: 2025 Tam Kılavuz

29 Ekim 2025 • 15 dakika okuma

Port yönlendirme, ev ağ yapılandırmasında yeni başlayanlar için en çok talep edilen ve aynı zamanda en zor konulardan biridir. Bu kılavuzda, port yönlendirmeyi herhangi bir amaç için doğru şekilde nasıl kuracağınızı ayrıntılı olarak tartışacağız: oyun sunucularından izleme sistemlerine kadar.

🔍 Port Yönlendirme Nedir?

Port Yönlendirme, ağ isteklerini router'ın harici portundan, yerel ağınızdaki bir cihazın belirli bir portuna yönlendirmenize olanak tanıyan bir teknolojidir.

⚠️ Anlaşılması Önemli

Port yönlendirme, cihazınıza internetten erişim açar. Her zaman güvenlik önlemlerini takip edin ve gereksiz portlar açmayın!

🔧 Kurulum Hazırlığı

  1. Cihazınızın statik IP adresini belirleyin

    Cihazınızın (PC, sunucu, kamera) kalıcı bir yerel IP adresine ihtiyacı vardır, böylece router trafiği nereye yönlendireceğini bilir.

    Windows'ta IP adresi nasıl bulunur:

    ipconfig

    Ağ bağlantınız için "IPv4 Address" satırını arayın

    Linux/macOS'ta IP adresi nasıl bulunur:

    ifconfig

    Veya şunu kullanın: ip addr show

  2. Statik IP yapılandırın

    IP adresinin değişmesini önlemek için statik olarak yapılandırın:

    • Router'da: DHCP rezervasyonu aracılığıyla statik IP atayın
    • Cihazda: Ağ ayarlarında manuel olarak statik IP yapılandırın

  3. Harici IP adresinizi öğrenin

    Bu, diğer kullanıcıların internet üzerinde gördüğü adrestir:

    Harici IP'yi öğrenme yolları:

    # Komut satırı aracılığıyla
curl ifconfig.me

# Veya şu web sitelerini ziyaret edin:
# - whatismyipaddress.com
# - 2ip.ru
# - myip.com

  4. Gerekli portları belirleyin

    Uygulamanızın veya hizmetinizin hangi portları kullandığını öğrenin. Bu bilgi genellikle belgelerde bulunur.

🚀 Adım Adım Kurulum Talimatları

  1. Router kontrol paneline giriş yapın

    Tarayıcınızı açın ve router'ın IP adresini girin. Genellikle şunlardır:

    • 192.168.1.1 (TP-Link, D-Link)
    • 192.168.0.1 (Netgear, bazı TP-Link)
    • 192.168.31.1 (Xiaomi)
    • 192.168.10.1 (Huawei)

    Kullanıcı adı/şifre genellikle router üzerindeki etikette listelenir.

  2. Port yönlendirme bölümünü bulun

    Bölüm farklı isimlere sahip olabilir:

    • Port Forwarding
    • Virtual Servers
    • Port Yönlendirme
    • NAT
    • Güvenlik Duvarı

  3. Port yönlendirme parametrelerini doldurun

    Şunları belirtmeniz gerekir:

    • Hizmet Adı: Herhangi bir isim (ör. "Minecraft Sunucusu")
    • Harici Port: Router'da açılacak port
    • Dahili Port: Cihazınızdaki port
    • Dahili IP: Cihazınızın statik IP'si
    • Protokol: TCP, UDP veya Her İkisi

  4. Ayarları kaydedin

    Tüm alanları doldurduktan sonra "Kaydet" veya "Uygula"ya basın. Router yeniden başlayabilir.

📡 Popüler Router'lar İçin Talimatlar

🔄 TP-Link

  1. Kontrol paneline giriş yapın (192.168.1.1)
  2. Şuraya gidin: Advanced → NAT Forwarding → Virtual Servers
  3. "Add New"a tıklayın
  4. Doldurun:
    • Service Type: Listeden seçin veya kendiniz girin
    • External Port: Dışarıdan erişim için port
    • Internal Port: Cihazdaki port
    • IP Address: Cihazın yerel IP'si
    • Protocol: TCP/UDP/BOTH
  5. Kaydedin ve router'ı yeniden başlatın

🛡️ ASUS

  1. Panele giriş yapın (192.168.1.1)
  2. Şuraya gidin: Advanced Settings → WAN → Virtual Server/Port Forwarding
  3. "Enable Port Forwarding"ı etkinleştirin
  4. Alanları doldurun:
    • Service Name: Hizmet adı
    • Port Range: Port aralığı (ör. 25565-25565)
    • Local IP: Cihazınızın IP'si
    • Local Port: Cihazınızın portu
    • Protocol: TCP/UDP/BOTH
  5. "Add" ve ardından "Apply"a tıklayın

🔗 D-Link

  1. Panele giriş yapın (192.168.0.1)
  2. Şuraya gidin: Advanced → Port Forwarding
  3. "Add"a tıklayın
  4. Doldurun:
    • Rule Name: Kural adı
    • External Port: Harici port
    • Internal Port: Dahili port
    • Internal IP: Cihazın IP'si
    • Schedule: Always
    • Inbound Filter: Allow All
  5. Ayarları kaydedin

📶 Zyxel Keenetic

  1. Panele giriş yapın (my.keenetic.net veya 192.168.1.1)
  2. Şuraya gidin: Güvenlik → Port Yönlendirme
  3. "Kural ekle"ye tıklayın
  4. Doldurun:
    • Açıklama: Hizmet adı
    • Arabirim: İnternet
    • Protokol: TCP/UDP
    • Harici port: Dışarıdan erişim için port
    • Hedef IP: Cihazın yerel IP'si
    • Dahili port: Cihazdaki port
  5. Değişiklikleri kaydedin

✅ Port Yönlendirme Çalışmasını Doğrulama

  1. Çevrimiçi hizmetleri kullanın

    Açık portları kontrol etmek için web siteleri:

    • portchecker.co
    • canyouseeme.org

  2. Komut satırı üzerinden doğrulama

    Başka bir bilgisayardan kontrol edin:

    # telnet veya nc (netcat) kullanın 
telnet HARICI-IP'NIZ port 
# Veya nmap kullanarak 
nmap -p port HARICI-IP'NIZ

  3. Hizmet testi

    Yerel ağınızın dışındaki başka bir cihazdan (örn. mobil internet üzerinden) hizmetinize bağlanmayı deneyin.

✅ Başarılı Kurulum İşaretleri

  • Port, çevrimiçi denetleyicilerde "açık" olarak görüntülenir
  • Dışarıdan hizmete bağlanabilirsiniz
  • Hizmet kesintisiz olarak kararlı şekilde çalışır

🔧 Yaygın Sorunları Giderme

🚫 Port Hala Kapalı

  • Güvenlik duvarını kontrol edin: Windows Güvenlik Duvarı'nda veya başka bir güvenlik duvarında istisna ekleyin
  • Router'ı yeniden başlatın: Bazen ayarlar yalnızca yeniden başlatmadan sonra uygulanır
  • IP adresini kontrol edin: Cihazın doğru statik IP'ye sahip olduğundan emin olun
  • UPnP'yi devre dışı bırakın: Bazen UPnP, manuel port ayarlarıyla çakışır

⚠️ Sağlayıcının Dinamik IP'si

Dinamik IP'niz varsa, değişebilir. Çözümler:

  • DDNS kullanın: No-IP, DuckDNS gibi hizmetler
  • IP'yi kontrol edin: Geçerli IP hakkındaki bilgileri düzenli olarak güncelleyin
  • Statik IP: Sağlayıcıdan statik IP sipariş edin (ücretli)

🔒 Çift NAT

Ağınızda birden fazla router olduğunda sorun ortaya çıkar:

  • Köprü modu: Router'lardan birini köprü moduna ayarlayın
  • DMZ: Ana router'da ikinci router için DMZ yapılandırın
  • Her ikisinde de yönlendirme: Her iki router'da da yönlendirmeyi yapılandırın

🛡️ Güvenlik Önlemleri

🔐 Kritik Kurallar

  1. Yalnızca gerekli portları açın - her açık port saldırı yüzeyini artırır
  2. Dışarıdan erişilebilen tüm hizmetlerde karmaşık şifreler kullanın
  3. Yazılımı düzenli olarak güncelleyin - güvenlik açıklarını ortadan kaldırın
  4. Router günlüklerini kullanarak bağlantıları izleyin
  5. Port yönlendirmeye daha güvenli bir alternatif olarak VPN'yi düşünün

🎯 En İyi Uygulamalar

  • Tarayıcıların dikkatini azaltmak için standart olmayan portlar kullanın
  • Portların programa göre otomatik olarak devre dışı bırakılmasını ayarlayın
  • Şüpheli etkinlik için günlükleri düzenli olarak kontrol edin
  • Şüpheli IP'leri engellemek için fail2ban veya benzerini kullanın
