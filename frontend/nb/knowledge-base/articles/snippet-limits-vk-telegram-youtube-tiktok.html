Tegnbegrensninger og utdragsklipping (VK / Telegram / YouTube / TikTok)

15. september 2025
Hvorfor lese dette: for å sikre at titler og beskrivelser ikke blir kuttet av og gir maksimal CTR. Nedenfor — sammendragstabell over grenser og visuelle før/etter-eksempler for retting.

1) Sammendragstabell over grenser og anbefalinger

Merk: plattformer endrer ofte grensesnitt og klipping. Vi gir testede anbefalte lengder og kjente harde grenser.

Plattform Tittel (anbef./grense) Beskrivelse/tekst (anbef./grense) Forhåndsvisning/klipping
VK ≤ 60–70 / — ≤ 140–160 / opptil 4096 tegn i innlegget Lenkeutdrag kutter tittel ~70, beskrivelse ~140+
Telegram ≤ 70 / — (hentet fra OG) ≤ 120–160 / 4096 i melding Forhåndsvisning fra OG: tittel ~70, beskrivelse ~120–140
YouTube ≤ 60 / grense 100 ≤ 150–200 (første linjer) / grense 5000 Above‑the‑fold viser ~2–3 linjer
TikTok ≤ 60–80 / — ≤ 80–150 (synlig del) / grense opptil ~2200 Bildetekst klippes, 'Mer' etter ~80–150

2) VK: før/etter

FØR

"En utrolig detaljert og veldig lang tittel om vår nye funksjon som uunngåelig vil bli kuttet i alle forhåndsvisninger og redusere CTR..."

ETTER

"Ny funksjon: raskere og enklere" — kort og klart. Undertittel/beskrivelse: "Vi har samlet nøkkelfunksjoner og brukseksempler" (≤ 140).

3) Telegram: før/etter

FØR

"[Ny artikkel] Komplett guide til UTM-merker for alle markedsføringskanaler, med lister, eksempler, sjekkliste og innholdsfortegnelse"

ETTER

"UTM: komplett guide" — "GA4, maler for Google/Meta/VK, vanlige feil". Tittel ≤ 70, beskrivelse ≤ 120–140.

4) YouTube: før/etter

FØR

"Hvordan vi bygde SEO på 30 dager: fullstendig oppdeling av alle trinn fra revisjon til implementering, analyse og rapportering [2025]" (105 tegn)

ETTER

"SEO på 30 dager: trinnvis plan" — "Revisjon, implementering, rapportering. Feil og life hacks (2025)". Tittel ≤ 60, første 150–200 tegn av beskrivelse — nøkkelfordeler og CTA.

5) TikTok: før/etter

FØR

"Se vår nye lange forklaring med masse detaljer og lenker — husk å lese til slutten!"

ETTER

"3 UTM-feil som spiser trafikk" — "Sjekk source/medium, koding og omdirigeringer. Komplett guide — via lenke".

6) Tips for forhåndsvisninger og metadata

  • Sett OG-meta (title/description/image) for korrekte forhåndsvisninger i VK/Telegram.
  • Hold title ≤ 60–70, description ≤ 140–160 for universell synlighet.
  • Plasser nøkkelfordeler i de første 1–2 linjene av beskrivelsen.
  • Sjekk forhåndsvisninger før publisering (via utkast/forhåndsvisningsbott/deling).
