Port Forwarding: Kompletan Vodič 2025

29. oktobar 2025. • 15 minuta čitanja

Port forwarding je jedna od najtraženijih, a istovremeno i najtežih tema za početnike u konfiguraciji kućne mreže. U ovom vodiču ćemo detaljno objasniti kako pravilno podesiti port forwarding za bilo koju svrhu: od servera za igre do sistema za nadzor.

Šta je Port Forwarding?

Port Forwarding je tehnologija koja omogućava preusmeravanje mrežnih zahteva sa spoljnog porta rutera na određeni port uređaja u vašoj lokalnoj mreži.

Internet

Spoljni zahtevi stižu do vašeg rutera

Ruter

Preusmerava zahteve na potrebni uređaj

Važno je razumeti

Port forwarding otvara pristup vašem uređaju sa interneta. Uvek poštujte bezbednosne mere i ne otvarajte nepotrebne portove!

Priprema za podešavanje

  1. Odredite statičku IP adresu uređaja

    Vaš uređaj (PC, server, kamera) treba da ima stalnu lokalnu IP adresu kako bi ruter znao gde da prosledi saobraćaj.

    Kako saznati IP adresu u Windows-u:

    ipconfig

    Potražite liniju "IPv4 Address" za vašu mrežnu vezu

    Kako saznati IP adresu u Linux/macOS:

    ifconfig

    Ili koristite: ip addr show

  2. Konfigurišite statičku IP

    Da biste sprečili promene IP adrese, konfigurišite je statički:

    • U ruteru: Postavite statičku IP preko DHCP rezervacije
    • Na uređaju: Konfigurišite statičku IP ručno u mrežnim podešavanjima

  3. Saznajte svoju spoljnu IP adresu

    Ovo je adresa koju drugi korisnici vide na internetu:

    Načini da saznate spoljnu IP:

    # Preko komandne linije
curl ifconfig.me

# Ili posetite sajtove:
# - whatismyipaddress.com
# - 2ip.ru
# - myip.com

  4. Odredite potrebne portove

    Saznajte koje portove koristi vaša aplikacija ili servis. Ove informacije se obično nalaze u dokumentaciji.

Korak-po-korak uputstva za podešavanje

  1. Prijavite se u kontrolni panel rutera

    Otvorite pretraživač i unesite IP adresu rutera. Obično je to:

    • 192.168.1.1 (TP-Link, D-Link)
    • 192.168.0.1 (Netgear, neki TP-Link)
    • 192.168.31.1 (Xiaomi)
    • 192.168.10.1 (Huawei)

    Korisničko ime/lozinka su obično navedene na nalepnici na ruteru.

  2. Pronađite odeljak za port forwarding

    Odeljak može imati različita imena:

    • Port Forwarding
    • Virtual Servers
    • Prosleđivanje Portova
    • NAT
    • Firewall

  3. Popunite parametre za port forwarding

    Morate da navedete:

    • Naziv Servisa: Proizvoljno ime (npr. "Minecraft Server")
    • Spoljni Port: Port koji će biti otvoren na ruteru
    • Interni Port: Port na vašem uređaju
    • Interni IP: Statička IP vašeg uređaja
    • Protokol: TCP, UDP ili Oba

  4. Sačuvajte podešavanja

    Nakon popunjavanja svih polja, pritisnite "Sačuvaj" ili "Primeni". Ruter se može ponovo pokrenuti.

Uputstva za popularne rutere

TP-Link

  1. Prijavite se u kontrolni panel (192.168.1.1)
  2. Idite na: Advanced → NAT Forwarding → Virtual Servers
  3. Kliknite "Add New"
  4. Popunite:
    • Service Type: Izaberite sa liste ili unesite svoj
    • External Port: Port za pristup spolja
    • Internal Port: Port na uređaju
    • IP Address: Lokalna IP uređaja
    • Protocol: TCP/UDP/BOTH
  5. Sačuvajte i ponovo pokrenite ruter

ASUS

  1. Prijavite se u panel (192.168.1.1)
  2. Idite na: Advanced Settings → WAN → Virtual Server/Port Forwarding
  3. Omogućite "Enable Port Forwarding"
  4. Popunite polja:
    • Service Name: Naziv servisa
    • Port Range: Opseg portova (npr. 25565-25565)
    • Local IP: IP vašeg uređaja
    • Local Port: Port vašeg uređaja
    • Protocol: TCP/UDP/BOTH
  5. Kliknite "Add" pa "Apply"

D-Link

  1. Prijavite se u panel (192.168.0.1)
  2. Idite na: Advanced → Port Forwarding
  3. Kliknite "Add"
  4. Popunite:
    • Rule Name: Naziv pravila
    • External Port: Spoljni port
    • Internal Port: Interni port
    • Internal IP: IP uređaja
    • Schedule: Always
    • Inbound Filter: Allow All
  5. Sačuvajte podešavanja

Zyxel Keenetic

  1. Prijavite se u panel (my.keenetic.net ili 192.168.1.1)
  2. Idite na: Bezbednost → Prosleđivanje Portova
  3. Kliknite "Dodaj pravilo"
  4. Popunite:
    • Opis: Naziv servisa
    • Interfejs: Internet
    • Protokol: TCP/UDP
    • Spoljni port: Port za pristup spolja
    • IP odredišta: Lokalna IP uređaja
    • Interni port: Port na uređaju
  5. Sačuvajte izmene

Provera rada port forwarding-a

  1. Koristite online servise

    Sajtovi za proveru otvorenih portova:

    • portchecker.co
    • canyouseeme.org

  2. Provera preko komandne linije

    Proverite sa drugog računara:

    # Koristite telnet ili nc (netcat) 
telnet VAŠ-SPOLJNI-IP port 
# Ili koristeći nmap 
nmap -p port VAŠ-SPOLJNI-IP

  3. Testiranje servisa

    Pokušajte da se povežete sa vašim servisom sa drugog uređaja van lokalne mreže (npr. preko mobilnog interneta).

Znaci uspešnog podešavanja

  • Port se prikazuje kao "otvoren" u online proverama
  • Možete se povezati sa servisom spolja
  • Servis radi stabilno bez prekida

Rešavanje uobičajenih problema

Port je i dalje zatvoren

  • Proverite firewall: Dodajte izuzetak u Windows Firewall ili drugi firewall
  • Ponovo pokrenite ruter: Ponekad se podešavanja primene tek nakon ponovnog pokretanja
  • Proverite IP adresu: Uverite se da uređaj ima tačnu statičku IP
  • Onemogućite UPnP: Ponekad UPnP dolazi u sukob sa ručnim podešavanjima portova

Dinamička IP od provajdera

Ako imate dinamičku IP, ona se može menjati. Rešenja:

  • Koristite DDNS: Servisi kao što su No-IP, DuckDNS
  • Proverite IP: Redovno ažurirajte informacije o trenutnoj IP
  • Statička IP: Poručite statičku IP od provajdera (plaćeno)

Dupli NAT

Problem se javlja kada imate više rutera u mreži:

  • Režim mosta: Podesite jedan od rutera u režim mosta
  • DMZ: Podesite DMZ na glavnom ruteru za drugi ruter
  • Prosleđivanje na oba: Podesite prosleđivanje na oba rutera

Bezbednosne mere

Kritična pravila

  1. Otvarajte samo potrebne portove - svaki otvoreni port povećava površinu napada
  2. Koristite složene lozinke na svim servisima dostupnim spolja
  3. Redovno ažurirajte softver - otklonite ranjivosti
  4. Nadgledajte konekcije koristeći logove rutera
  5. Razmotrite VPN kao sigurniju alternativu port forwarding-u

Najbolje prakse

  • Koristite nestandardne portove da smanjite pažnju skenera
  • Podesite automatsko onemogućavanje portova prema rasporedu
  • Proveravajte logove redovno za sumnjivu aktivnost
  • Koristite fail2ban ili slično za blokiranje sumnjivih IP-ova
