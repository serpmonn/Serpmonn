Comment jouer aux Carrés bleus tombants ?

  • Contrôles : Utilisez les flèches ou le tactile pour déplacer le carré rouge
  • Objectif : Esquiver les objets qui tombent
  • Points : 1 point par objet esquivé
  • Ratés : Si un objet vous touche, c'est un raté
  • Fin de partie : La partie se termine après 10 collisions
  • Classement : Après la partie, consultez vos résultats
  • Pause : Utilisez le bouton Pause pour arrêter temporairement
