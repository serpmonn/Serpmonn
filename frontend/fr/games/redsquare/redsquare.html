Comment jouer à Évasion carrée ?

1. Contrôlez le carré rouge avec les boutons, flèches ou glissements sur mobile

2. Évitez les collisions avec les carrés bleus et obstacles verts

3. Collectez les bonus dorés pour des points supplémentaires

4. Plus vous survivez longtemps, plus vous gagnez de points

5. Utilisez le bouton Pause (ou espace) pour faire une pause
