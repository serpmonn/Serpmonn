পোর্ট ফরওয়ার্ডিং: সম্পূর্ণ গাইড ২০২৫

২৯ অক্টোবর ২০২৫ • ১৫ মিনিট পড়া

পোর্ট ফরওয়ার্ডিং হোম নেটওয়ার্ক কনফিগারেশনের সবচেয়ে চাহিদাসম্পন্ন এবং একই সাথেদের জন্য সবচেয়ে কঠিন বিষয়গুলির মধ্যে একটি। এই গাইডে, আমরা বিস্তারিতভাবে আলোচনা করব কীভাবে যেকোনো উদ্দেশ্যে পোর্ট ফরওয়ার্ডিং সঠিকভাবে সেট আপ করতে হয়: গেম সার্ভার থেকে পর্যবেক্ষণ সিস্টেম পর্যন্ত।

পোর্ট ফরওয়ার্ডিং কি?

পোর্ট ফরওয়ার্ডিং (Port Forwarding) হল একটি প্রযুক্তি যা নেটওয়ার্ক অনুরোধগুলিকে রাউটারের বাহ্যিক পোর্ট থেকে আপনার লোকাল নেটওয়ার্কের একটি ডিভাইসের নির্দিষ্ট পোর্টে পুনঃনির্দেশিত করতে দেয়।

বোঝা গুরুত্বপূর্ণ

পোর্ট ফরওয়ার্ডিং ইন্টারনেট থেকে আপনার ডিভাইসে অ্যাক্সেস খুলে দেয়। সর্বদা নিরাপত্তা ব্যবস্থা মেনে চলুন এবং অপ্রয়োজনীয় পোর্ট খুলবেন না!

সেটআপের জন্য প্রস্তুতি

  1. ডিভাইসের স্ট্যাটিক IP ঠিকানা নির্ধারণ করুন

    আপনার ডিভাইসের (PC, সার্ভার, ক্যামেরা) একটি স্থায়ী লোকাল IP ঠিকানা প্রয়োজন যাতে রাউটার জানতে পারে ট্রাফিক কোথায় ফরওয়ার্ড করতে হবে।

    Windows-এ IP ঠিকানা কীভাবে খুঁজে পাবেন:

    ipconfig

    আপনার নেটওয়ার্ক সংযোগের জন্য "IPv4 Address" লাইনটি খুঁজুন

    Linux/macOS-এ IP ঠিকানা কীভাবে খুঁজে পাবেন:

    ifconfig

    অথবা ব্যবহার করুন: ip addr show

  2. স্ট্যাটিক IP কনফিগার করুন

    IP ঠিকানা পরিবর্তন হওয়া রোধ করতে, এটি স্ট্যাটিকভাবে কনফিগার করুন:

    • রাউটারে: DHCP রিজার্ভেশন এর মাধ্যমে স্ট্যাটিক IP নির্ধারণ করুন
    • ডিভাইসে: নেটওয়ার্ক সেটিংসে ম্যানুয়ালি স্ট্যাটিক IP কনফিগার করুন

  3. বাহ্যিক IP ঠিকানা জানুন

    এটি সেই ঠিকানা যা অন্যান্য ব্যবহারকারীরা ইন্টারনেটে দেখতে পায়:

    বাহ্যিক IP জানার উপায়:

    # কমান্ড লাইনের মাধ্যমে
curl ifconfig.me

# অথবা সাইটগুলি দেখুন:
# - whatismyipaddress.com
# - 2ip.ru
# - myip.com

  4. প্রয়োজনীয় পোর্টগুলি নির্ধারণ করুন

    আপনার অ্যাপ্লিকেশন বা পরিষেবা কোন পোর্টগুলি ব্যবহার করে তা জানুন। এই তথ্যটি সাধারণত ডকুমেন্টেশনে থাকে।

ধাপে ধাপে সেটআপ নির্দেশিকা

  1. রাউটারের নিয়ন্ত্রণ প্যানেলে লগইন করুন

    ব্রাউজার খুলুন এবং রাউটারের IP ঠিকানা লিখুন। সাধারণত এটি হয়:

    • 192.168.1.1 (TP-Link, D-Link)
    • 192.168.0.1 (Netgear, কিছু TP-Link)
    • 192.168.31.1 (Xiaomi)
    • 192.168.10.1 (Huawei)

    ইউজারনেম/পাসওয়ার্ড ডিফল্ট সাধারণত রাউটারের স্টিকার এ দেওয়া থাকে।

  2. পোর্ট ফরওয়ার্ডিং বিভাগটি খুঁজুন

    বিভাগটি ভিন্ন নামে হতে পারে:

    • Port Forwarding
    • Virtual Servers
    • পোর্ট ফরওয়ার্ডিং
    • NAT
    • ফায়ারওয়াল

  3. পোর্ট ফরওয়ার্ডিং প্যারামিটার পূরণ করুন

    আপনাকে নির্দিষ্ট করতে হবে:

    • সেবার নাম: ইচ্ছামত নাম (উদাহরণস্বরূপ "Minecraft Server")
    • বাহ্যিক পোর্ট: রাউটারে খোলা হবে এমন পোর্ট
    • অভ্যন্তরীণ পোর্ট: আপনার ডিভাইসের পোর্ট
    • অভ্যন্তরীণ IP: আপনার ডিভাইসের স্ট্যাটিক IP
    • প্রোটোকল: TCP, UDP বা উভয়

  4. সেটিংস সংরক্ষণ করুন

    সমস্ত ক্ষেত্র পূরণ করার পর "Save" বা "Apply" বাটনে ক্লিক করুন। রাউটার পুনরায় চালু হতে পারে।
