Omezení počtu znaků a ořezávání snippetů (VK / Telegram / YouTube / TikTok)

15. září 2025
Proč číst: aby se nadpisy a popisy neořezávaly a poskytovaly maximální CTR. Níže — souhrnná tabulka limitů a vizuální příklady «před/po» pro opravu.

1) Souhrnná tabulka limitů a doporučení

Poznámka: platformy často mění UI a ořezávání. Poskytujeme ověřené doporučené délky a známé pevné limity.

Platforma Nadpis (dopor./limit) Popis/text (dopor./limit) Náhled/ořezávání
VK ≤ 60–70 / — ≤ 140–160 / až 4096 znaků v příspěvku Link snippet ořezává nadpis ~70, popis ~140+
Telegram ≤ 70 / — (převzato z OG) ≤ 120–160 / 4096 ve zprávě Náhled z OG: nadpis ~70, popis ~120–140
YouTube ≤ 60 / limit 100 ≤ 150–200 (první řádky) / limit 5000 Above‑the‑fold zobrazuje ~2–3 řádky
TikTok ≤ 60–80 / — ≤ 80–150 (viditelná část) / limit až ~2200 Popisek se ořezává, «Více» po ~80–150

2) VK: «před/po»

PŘED

«Neuvěřitelně detailní a velmi dlouhý nadpis o naší nové funkci, který se nevyhnutelně ořeže ve všech náhledech a sníží CTR…»

PO

«Nová funkce: rychlejší a jednodušší» — krátké a jasné. Podnadpis/popis: «Shromáždili jsme klíčové možnosti a příklady použití» (≤ 140).

3) Telegram: «před/po»

PŘED

«[Nový článek] Kompletní průvodce UTM tagy pro všechny marketingové kanály, se seznamy, příklady, kontrolním seznamem a obsahem»

PO

«UTM: kompletní průvodce» — «GA4, šablony pro Google/Meta/VK, běžné chyby». Nadpis ≤ 70, popis ≤ 120–140.

4) YouTube: «před/po»

PŘED

«Jak jsme postavili SEO za 30 dní: kompletní rozbor všech kroků od auditu po implementaci, analýzy a reporting [2025]» (105 znaků)

PO

«SEO za 30 dní: krok za krokem plán» — «Audit, implementace, reporting. Chyby a life hacky (2025)». Nadpis ≤ 60, prvních 150–200 znaků popisu — klíčové výhody a CTA.

5) TikTok: «před/po»

PŘED

«Podívejte se na naše nové dlouhé vysvětlení se spoustou detailů a odkazů — nezapomeňte dočíst až do konce!»

PO

«3 chyby v UTM, které žerou traffic» — «Zkontrolujte source/medium, kódování a přesměrování. Kompletní průvodce — přes odkaz».

6) Tipy pro náhledy a metadata

  • Nastavte OG meta (title/description/image) pro správné náhledy ve VK/Telegramu.
  • Udržujte title ≤ 60–70, description ≤ 140–160 pro univerzální viditelnost.
  • Umístěte klíčovou výhodu do prvních 1–2 řádků popisu.
  • Zkontrolujte náhledy před publikací (přes koncepty/náhled boty/sdílení).
