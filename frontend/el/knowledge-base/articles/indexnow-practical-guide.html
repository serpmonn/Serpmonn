IndexNow: Πρακτική εφαρμογή και συμβουλές

15 Σεπτεμβρίου 2025 • 5 λεπτά ανάγνωσης

1) Τι είναι το IndexNow

Το IndexNow είναι ένα πρωτόκολλο που σας επιτρέπει να ειδοποιείτε αμέσως τις μηχανές αναζήτησης για αλλαγές URL (δημιουργία, ενημέρωση, διαγραφή). Υποστηρίζεται από Bing, Yandex, Seznam και άλλες υπηρεσίες. Το πλεονέκτημα είναι η ελάχιστη καθυστέρηση ευρετηρίασης και λιγότερο θόρυβο crawl.

2) Ρύθμιση κλειδιού και βασικό ping

  1. Δημιουργήστε ένα κλειδί (GUID) και τοποθετήστε το αρχείο YOUR_KEY.txt στη ρίζα του ιστότοπου (/.well-known/ ή ρίζα domain) με την συμβολοσειρά κλειδιού.
  2. Κάντε ping στο endpoint με URL και κλειδί. Παράδειγμα (μονό URL):
POST https://api.indexnow.org/indexnow Content-Type: application/json { "host": "www.serpmonn.ru", "key": "YOUR_KEY", "urlList": [ "https://www.serpmonn.ru/frontend/knowledge-base/articles/utm-complete-guide.html" ] }
Για παρτίδες, καθορίστε έως και 10.000 URL σε ένα αίτημα ή χρησιμοποιήστε μια τροφοδοσία αρχείου με URL αρχείου καταγραφής αλλαγών.

3) Λειτουργία μόνο-αλλαγμένο

Η ιδέα είναι να κάνετε ping μόνο τα αρχεία που έχουν πραγματικά αλλάξει στην τελευταία commit/ανάπτυξη. Αυτό μειώνει τον θόρυβο και τη χρήση ορίου.

  • Λάβετε τη λίστα των αλλαγμένων διαδρομών από VCS (git diff) ή τα αποτελέσματα κατασκευής.
  • Φιλτράρετε με δημόσιες διαδρομές (εξαιρώντας node_modules, .git, ιδιωτικούς φακέλους).
  • Ομαδοποιήστε ανά domain/γλώσσες για να αποφύγετε την αποστολή άσχετων URL.
Στην παραγωγή, είναι χρήσιμο να έχετε εφεδρική λύση: εάν το μόνο-αλλαγμένο είναι κενό (π.χ. επείγουσα διόρθωση για ρυθμίσεις), κάντε ping σε σημαντικούς χάρτες ιστότοπου.

4) IndexNow vs sitemap ping

  • Ταχύτητα: Το IndexNow παραδίδει ενημερώσεις γρηγορότερα, δεν περιμένει τον κύκλο crawl· το sitemap ping σηματοδοτεί την παρουσία ενός sitemap αλλά όχι αλλαγές σε ένα συγκεκριμένο URL.
  • Θόρυβος: IndexNow - στοχευμένα σε αλλαγμένα URL· sitemap - σάρωση ολόκληρου του χάρτη (πιο ακριβό για τον προϋπολογισμό crawl).
  • Αξιοπιστία: Τα sitemaps παραμένουν η βάση· το IndexNow συμπληρώνει και επιταχύνει. Είναι καλύτερα να χρησιμοποιείτε και τα δύο.

5) Κοινά λάθη

  • Το κλειδί δεν είναι προσβάσιμο μέσω δημόσιου URL ή δεν ταιριάζει με την παράμετρο key.
  • Ping σε ιδιωτικές/404 σελίδες, παραμέτρους χωρίς κανονικοποίηση.
  • Πολύ συχνές επαναλήψεις των ίδιων URL (περιορίστε τις επαναλήψεις).
  • Αγνοώντας την πολυγλωσσία (πρέπει να κάνετε ping την αντίστοιχη εκδοχή).

6) Συχνές ερωτήσεις και λίστα ελέγχου

Πόσα URL να στείλω ταυτόχρονα;

Έως 10.000 σε μια παρτίδα - βέλτιστα ομαδοποιημένα ανά τύπο/γλώσσα.

Πρέπει να κάνω ping διαγραφές;

Ναι - μεταβιβάστε URL διαγραμμένων σελίδων για να επιταχύνετε την απευρετηρίαση.

Πώς να επιβεβαιώσω ότι λειτουργεί;

Αρχεία καταγραφής απόκρισης API (200/202), αιχμές crawl στα αρχεία καταγραφής διακομιστή, ταχύτερη εμφάνιση νέων URL.

  • Το κλειδί τοποθετήθηκε και είναι προσβάσιμο μέσω HTTPS.
  • Συλλογή μόνο-αλλαγμένων διαδρομών από CI/CD.
  • Αποδιπλοποίηση και περιορισμός επαναλήψεων.
  • Εφεδρική λύση σε sitemap ping σε κενή παρτίδα.
: Το IndexNow επιταχύνει σημαντικά την ευρετηρίαση νέων και τροποποιημένων σελίδων. Σε συνδυασμό με τα παραδοσιακά sitemaps, είναι ένα ισχυρό εργαλείο για βελτιστοποίηση SEO.
