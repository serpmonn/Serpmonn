Περιορισμοί χαρακτήρων και περικοπή snippet (VK / Telegram / YouTube / TikTok)

15 Σεπτεμβρίου 2025
Γιατί να το διαβάσετε: για να μην περικόπτονται οι τίτλοι και οι περιγραφές και να παρέχουν μέγιστο CTR. Παρακάτω — συνοπτικός πίνακας ορίων και οπτικά παραδείγματα «πριν/μετά» για διόρθωση.

1) Συνοπτικός πίνακας ορίων και συστάσεων

Σημείωση: οι πλατφόρμες αλλάζουν συχνά UI και περικοπή. Παροχυμε δοκιμασμένες προτεινόμενες μήκη και γνωστά αυστηρά όρια.

Πλατφόρμα Τίτλος (προτ./όριο) Περιγραφή/κείμενο (προτ./όριο) Προεπισκόπηση/περικοπή
VK ≤ 60–70 / — ≤ 140–160 / έως 4096 χαρακτήρες στη δημοσίευση Το snippet συνδέσμου περικόπτει τίτλο ~70, περιγραφή ~140+
Telegram ≤ 70 / — (από OG) ≤ 120–160 / 4096 στο μήνυμα Προεπισκόπηση από OG: τίτλος ~70, περιγραφή ~120–140
YouTube ≤ 60 / όριο 100 ≤ 150–200 (πρώτες γραμμές) / όριο 5000 Above‑the‑fold εμφανίζει ~2–3 γραμμές
TikTok ≤ 60–80 / — ≤ 80–150 (ορατό μέρος) / όριο έως ~2200 Η λεζάντα περικόπτεται, «Περισσότερα» μετά από ~80–150

2) VK: «πριν/μετά»

ΠΡΙΝ

«Απίστευτα λεπτομερής και πολύ μεγάλος τίτλος για τη νέα μας λειτουργία, που αναπόφευκτα θα περικοπεί σε όλες τις προεπισκοπήσεις και θα μειώσει το CTR…»

ΜΕΤΑ

«Νέα λειτουργία: πιο γρήγορη και απλή» — σύντομη και σαφής. Υπότιτλος/περιγραφή: «Συλλέξαμε τις κύριες δυνατότητες και παραδείγματα χρήσης» (≤ 140).

3) Telegram: «πριν/μετά»

ΠΡΙΝ

«[Νέο άρθρο] Πλήρης οδηγός για ετικέτες UTM για όλα τα κανάλια μάρκετινγκ, με λίστες, παραδείγματα, checklist και πίνακα περιεχομένων»

ΜΕΤΑ

«UTM: πλήρης οδηγός» — «GA4, πρότυπα για Google/Meta/VK, συχνά λάθη». Τίτλος ≤ 70, περιγραφή ≤ 120–140.

4) YouTube: «πριν/μετά»

ΠΡΙΝ

«Πώς χτίσαμε SEO σε 30 ημέρες: πλήρης ανάλυση όλων των βημάτων από έλεγχο έως εφαρμογή, ανάλυση και αναφορά [2025]» (105 χαρακτήρες)

ΜΕΤΑ

«SEO σε 30 ημέρες: σχέδιο βήμα προς βήμα» — «Έλεγχος, εφαρμογή, αναφορά. Λάθη και life hacks (2025)». Τίτλος ≤ 60, πρώτοι 150–200 χαρακτήρες περιγραφής — βασικά οφέλη και CTA.

5) TikTok: «πριν/μετά»

ΠΡΙΝ

«Δείτε τη νέα μας μακρά εξήγηση με πολλές λεπτομέρειες και συνδέσμους — φροντίστε να διαβάσετε μέχρι το τέλος!»

ΜΕΤΑ

«3 λάθη στο UTM που καταναλώνουν κίνηση» — «Ελέγξτε source/medium, κωδικοποίηση και ανακατευθύνσεις. Πλήρης οδηγός — μέσω συνδέσμου».

6) Συμβουλές για προεπισκοπήσεις και μεταδεδομένα

  • Ορίστε OG μεταδεδομένα (title/description/image) για σωστές προεπισκοπήσεις στο VK/Telegram.
  • Κρατήστε title ≤ 60–70, description ≤ 140–160 για καθολική ορατότητα.
  • Τοποθετήστε το βασικό όφελος στις πρώτες 1–2 γραμμές της περιγραφής.
  • Ελέγξτε τις προεπισκοπήσεις πριν από τη δημοσίευση (μέσω προσχεδίων/προεπισκόπησης bots/κοινής χρήσης).
