How to play Falling Blue Squares?

  • Controls: Use arrow keys or touch on a touchscreen to move the red square left and right
  • Goal: Dodge falling objects
  • Scoring: You get 1 point for each object you dodge
  • Misses: If an object hits you, it counts as a miss
  • Game over: The game ends after 10 collisions
  • Leaderboard: After the game you can view your results on the leaderboard
  • Pause: Use the Pause button to temporarily stop the game
