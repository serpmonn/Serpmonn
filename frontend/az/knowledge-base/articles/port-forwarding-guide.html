Port İstiqamətləndirmə: Tam Bələdçi 2025

29 oktyabr 2025 • 15 dəqiqə oxu

Port istiqamətləndirmə ev şəbəkə konfiqurasiyasında ən tələb olunan və eyni zamanda yeni başlayanlar üçün ən çətin mövzulardan biridir. Bu bələdçidə, hər hansı məqsəd üçün port istiqamətləndirməni necə düzgün quraşdırmağı ətraflı izah edəcəyik: oyun serverlərindən müşahidə sistemlərinə qədər.

🔍 Port İstiqamətləndirmə nədir?

Port İstiqamətləndirmə (Port Forwarding) şəbəkə sorğularını routerin xarici portundan yerli şəbəkənizdəki cihazın müəyyən bir portuna yönləndirməyə imkan verən texnologiyadır.

⚠️ Anlamaq vacibdir

Port istiqamətləndirmə cihazınıza internetdən girişi açır. Həmişə təhlükəsizlik tədbirlərinə riayət edin və lazımsız portları açmayın!

🔧 Quraşdırma üçün Hazırlıq

  1. Cihazın statik IP ünvanını müəyyənləşdirin

    Cihazınızın (PC, server, kamera) routerin trafiki hara yönləndirəcəyini bilməsi üçün daimi yerli IP ünvanına ehtiyacı var.

    Windows-da IP ünvanını necə tapmaq olar:

    ipconfig

    Şəbəkə bağlantınız üçün "IPv4 Address" sətrini axtarın

    Linux/macOS-da IP ünvanını necə tapmaq olar:

    ifconfig

    Və ya istifadə edin: ip addr show

  2. Statik IP konfiqurasiya edin

    IP ünvanının dəyişməsinin qarşısını almaq üçün onu statik olaraq konfiqurasiya edin:

    • Routerdə: DHCP rezervasiyası vasitəsilə statik IP təyin edin
    • Cihazda: Şəbəkə parametrlərində əl ilə statik IP konfiqurasiya edin

  3. Xarici IP ünvanını öyrənin

    Bu, digər istifadəçilərin internetdə gördüyü ünvandır:

    Xarici IP-ni tapmaq üsulları:

    # Əmr sətri vasitəsilə
curl ifconfig.me

# Və ya saytları ziyarət edin:
# - whatismyipaddress.com
# - 2ip.ru
# - myip.com

  4. Tələb olunan portları müəyyənləşdirin

    Proqramınızın və ya xidmətinizin hansı portlardan istifadə etdiyini öyrənin. Bu məlumat adətən sənədləşmədə olur.

🚀 Addım-addım quraşdırma təlimatları

  1. Routerin idarəetmə panelinə daxil olun

    Brauzeri açın və routerin IP ünvanını daxil edin. Adətən bu:

    • 192.168.1.1 (TP-Link, D-Link)
    • 192.168.0.1 (Netgear, bəzi TP-Link)
    • 192.168.31.1 (Xiaomi)
    • 192.168.10.1 (Huawei)

    İstifadəçi adı/parol adətən routerin üzərindəki etiketdə göstərilir.

  2. Port istiqamətləndirmə bölməsini tapın

    Bölmə fərqli adlana bilər:

    • Port Forwarding
    • Virtual Servers
    • Port Yönləndirmə
    • NAT
    • Firewall

  3. Port istiqamətləndirmə parametrlərini doldurun

    Təyin etməli olduğunuz məlumatlar:

    • Xidmət Adı: İxtiyari ad (məsələn, "Minecraft Server")
    • Xarici Port: Routerdə açılacaq port
    • Daxili Port: Cihazınızdakı port
    • Daxili IP: Cihazınızın statik IP-si
    • Protokol: TCP, UDP və ya Hər ikisi

  4. Parametrləri saxlayın

    Bütün sahələri doldurduqdan sonra "Save" və ya "Apply" düyməsini basın. Router yenidən yüklənə bilər.

📡 Məşhur routerlər üçün təlimatlar

🔄 TP-Link

  1. İdarəetmə panelinə daxil olun (192.168.1.1)
  2. Keçin: Advanced → NAT Forwarding → Virtual Servers
  3. "Add New" düyməsini basın
  4. Doldurun:
    • Service Type: Siyahıdan seçin və ya özünüz daxil edin
    • External Port: Xarici giriş üçün port
    • Internal Port: Cihazdakı port
    • IP Address: Cihazın yerli IP-si
    • Protocol: TCP/UDP/BOTH
  5. Saxlayın və routeri yenidən yükləyin

🛡️ ASUS

  1. Panelə daxil olun (192.168.1.1)
  2. Keçin: Advanced Settings → WAN → Virtual Server/Port Forwarding
  3. "Enable Port Forwarding" aktiv edin
  4. Sahələri doldurun:
    • Service Name: Xidmət adı
    • Port Range: Port diapazonu (məsələn, 25565-25565)
    • Local IP: Cihazınızın IP-si
    • Local Port: Cihaz portu
    • Protocol: TCP/UDP/BOTH
  5. "Add" və "Apply" düymələrini basın

🔗 D-Link

  1. Panelə daxil olun (192.168.0.1)
  2. Keçin: Advanced → Port Forwarding
  3. "Add" düyməsini basın
  4. Doldurun:
    • Rule Name: Qayda adı
    • External Port: Xarici port
    • Internal Port: Daxili port
    • Internal IP: Cihaz IP-si
    • Schedule: Always
    • Inbound Filter: Allow All
  5. Parametrləri saxlayın

📶 Zyxel Keenetic

  1. Panelə daxil olun (my.keenetic.net və ya 192.168.1.1)
  2. Keçin: Təhlükəsizlik → Port İstiqamətləndirmə
  3. "Qayda əlavə et" düyməsini basın
  4. Doldurun:
    • Təsvir: Xidmət adı
    • İnterfeys: İnternet
    • Protokol: TCP/UDP
    • Xarici port: Xarici giriş üçün port
    • Təyinat IP-si: Cihazın yerli IP-si
    • Daxili port: Cihazdakı port
  5. Dəyişiklikləri saxlayın

✅ Port istiqamətləndirmə işinin yoxlanılması

  1. Onlayn xidmətlərdən istifadə edin

    Açıq portları yoxlamaq üçün saytlar:

    • portchecker.co
    • canyouseeme.org
    • 2ip.ru/port-scan/

  2. Əmr sətri vasitəsilə yoxlama

    Başqa kompüterdən yoxlama:

    # telnet və ya nc (netcat) istifadə edin 
telnet sizin-xarici-ip port 
# Və ya nmap istifadə edərək 
nmap -p port sizin-xarici-ip

  3. Xidmətin sınaqdan keçirilməsi

    Yerli şəbəkədən kənar başqa cihazdan (məsələn, mobil internet vasitəsilə) xidmətinizə qoşulmağa çalışın.

✅ Uğurlu quraşdırma əlamətləri

  • Port onlayn yoxlayıcıda "açıq" kimi göstərilir
  • Xaricdən xidmətə qoşula bilərsiniz
  • Xidmət kəsilmədən sabit işləyir

🔧 Ümumi problemlərin həlli

🚫 Port hələ də bağlıdır

  • Firewall-u yoxlayın: Windows Firewall və ya digər firewall-da istisna əlavə edin
  • Routeri yenidən yükləyin: Bəzən parametrlər yalnız yenidən yükləndikdən sonra tətbiq olunur
  • IP ünvanını yoxlayın: Cihazın düzgün statik IP-yə malik olduğundan əmin olun
  • UPnP-ni deaktiv edin: Bəzən UPnP əl ilə port parametrləri ilə ziddiyyət təşkil edir

⚠️ Provayderin dinamik IP-si

Dinamik IP-niz varsa, o dəyişə bilər. Həllər:

  • DDNS istifadə edin: No-IP, DuckDNS kimi xidmətlər
  • IP-ni yoxlayın: Cari IP haqqında məlumatı müntəzəm yeniləyin
  • Statik IP: Provayderdən statik IP sifariş edin (pullu)

🔒 İkiqat NAT

Problem şəbəkədə bir neçə router olduqda yaranır:

  • Körpü rejimi: Routerlərdən birini körpü rejiminə qurun
  • DMZ: Əsas routerdə ikinci router üçün DMZ qurun
  • Hər ikisində yönləndirmə: Hər iki routerdə yönləndirmə qurun

🛡️ Təhlükəsizlik tədbirləri

🔐 Kritik qaydalar

  1. Yalnız zəruri portları açın - hər açıq port hücum səthini artırır
  2. Xaricdən əlçatan bütün xidmətlərdə mürəkkəb parollardan istifadə edin
  3. Proqram təminatını müntəzəm yeniləyin - zəiflikləri aradan qaldırın
  4. Router jurnallarından istifadə edərək əlaqələri izləyin
  5. VPN-i nəzərdən keçirin port istiqamətləndirmə üçün daha təhlükəsiz alternativ kimi

🎯 Ən yaxşı təcrübələr

  • Skanerlərin diqqətini azaltmaq üçün qeyri-standart portlardan istifadə edin
  • Portların avtomatik deaktivasiyasını cədvəl üzrə qurun
  • Jurnalları şübhəli fəaliyyət üçün müntəzəm yoxlayın
  • Şübhəli IP-ləri bloklamaq üçün fail2ban və ya oxşar proqramdan istifadə edin
