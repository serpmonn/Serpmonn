Teckenbegränsningar och snippet-avklippning (VK / Telegram / YouTube / TikTok)

15 september 2025
Varför läsa detta: för att säkerställa att titlar och beskrivningar inte klipps av och ger maximal CTR. Nedan — sammanfattningstabell över begränsningar och visuella 'före/efter'-exempel för korrigering.

1) Sammanfattningstabell över begränsningar och rekommendationer

Obs: plattformar ändrar ofta gränssnitt och avklippning. Vi tillhandahåller testade rekommenderade längder och kända hårda begränsningar.

Plattform Titel (rek./gräns) Beskrivning/text (rek./gräns) Förhandsgranskning/avklippning
VK ≤ 60–70 / — ≤ 140–160 / upp till 4096 tecken i inlägget Länk-snippet klipper av titel ~70, beskrivning ~140+
Telegram ≤ 70 / — (hämtat från OG) ≤ 120–160 / 4096 i meddelande Förhandsgranskning från OG: titel ~70, beskrivning ~120–140
YouTube ≤ 60 / gräns 100 ≤ 150–200 (första raderna) / gräns 5000 Above‑the‑fold visar ~2–3 rader
TikTok ≤ 60–80 / — ≤ 80–150 (synlig del) / gräns upp till ~2200 Bildtext klipps av, 'Mer' efter ~80–150

2) VK: 'före/efter'

FÖRE

"En otroligt detaljerad och väldigt lång titel om vår nya funktion som oundvikligen kommer att klippas av i alla förhandsgranskningar och minska CTR…"

EFTER

„Ny funktion: snabbare och enklare” — kort och klart. Underrubrik/beskrivning: „Vi har samlat nyckelfunktioner och användningsexempel” (≤ 140).

3) Telegram: 'före/efter'

FÖRE

"[Ny artikel] Komplett guide för UTM-taggar för alla marknadsföringskanaler, med listor, exempel, checklista och innehållsförteckning"

EFTER

„UTM: komplett guide” — „GA4, mallar för Google/Meta/VK, vanliga misstag”. Titel ≤ 70, beskrivning ≤ 120–140.

4) YouTube: 'före/efter'

FÖRE

"Hur vi byggde SEO på 30 dagar: fullständig uppdelning av alla steg från granskning till implementering, analys och rapportering [2025]" (105 tecken)

EFTER

„SEO på 30 dagar: steg-för-steg-plan” — „Granskning, implementering, rapportering. Misstag och life hacks (2025)”. Titel ≤ 60, första 150–200 tecknen av beskrivning — nyckelfördelar och CTA.

5) TikTok: 'före/efter'

FÖRE

"Titta på vår nya långa förklaring med massor av detaljer och länkar — kom ihåg att läsa till slutet!"

EFTER

„3 UTM-misstag som äter upp trafik” — „Kolla source/medium, kodning och omdirigeringar. Komplett guide — via länk".

6) Tips för förhandsgranskning och metadata

  • Ställ in OG-meta (title/description/image) för korrekt förhandsgranskning i VK/Telegram.
  • Behåll title ≤ 60–70, description ≤ 140–160 för universell synlighet.
  • Placera nyckelfördelar i de första 1–2 raderna av beskrivningen.
  • Kolla förhandsgranskning innan publicering (via utkast/förhandsgranskningsbottar/delning).
