Ce este Serpmonn.ru?

Serpmonn este o platformă cuprinzătoare care se concentrează pe instrumente practice: autentificare și profile, știri RSS, jocuri browser cu tabele de lideri și roboți Telegram (jocuri, verificare, meme). O atenție deosebită este acordată securității: au fost adăugate endpoint-uri CSRF și limitarea ratei în serviciile principale. Scopul Serpmonn.ru este de a combina acestea într-o interfață convenabilă pentru sarcinile zilnice.

Proiectul se dezvoltă deschis: UX pe dispozitive mobile și desktop este îmbunătățit treptat, sunt adăugate instrumente, accesibilitatea este extinsă (font mare, contrast ridicat, mai puține animații). Planurile includ mai multe jocuri HTML5, căutare îmbunătățită, integrații și un sistem complet de profil.

Funcționalitățile Proiectului

Securitate

Proiectul este întărit treptat: sunt folosite token-uri PASETO, endpoint-urile CSRF sunt implementate, limitarea ratei este adăugată, dependențele tranzitive sunt actualizate. Se efectuează analiză statică (CodeQL) și actualizări regulate ale dependențelor.

Tehnologii

  • Backend: Node.js, Express, MySQL, PASETO, PM2.
  • Frontend: HTML, CSS, JavaScript, PWA.
  • Infrastructură: Nginx, CI/CD, CodeQL.

Planuri
