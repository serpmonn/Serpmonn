1) 제한 및 권장사항 요약표

참고: 플랫폼은 자주 UI 및 잘림을 변경합니다. 우리는 테스트된 권장 길이와 알려진 엄격한 제한을 제공합니다.

플랫폼 제목 (권장/제한) 설명/텍스트 (권장/제한) 미리보기/잘림
VK ≤ 60–70 / — ≤ 140–160 / 게시물에서 최대 4096자 링크 스니펫 제목 ~70자, 설명 ~140+자 잘림
Telegram ≤ 70 / — (OG에서 가져옴) ≤ 120–160 / 메시지에서 4096자 OG에서 미리보기: 제목 ~70자, 설명 ~120–140자
YouTube ≤ 60 / 제한 100자 ≤ 150–200 (첫 줄) / 제한 5000자 Above‑the‑fold 약 2–3줄 표시
TikTok ≤ 60–80 / — ≤ 80–150 (보이는 부분) / 최대 약 2200자 제한 캡션 잘림, '더 보기' 약 80–150자 후

2) VK: '이전/이후'

이전

"우리의 새로운 기능에 대한 믿을 수 없을 정도로 상세하고 매우 긴 제목은 모든 미리보기에서 불가피하게 잘리고 CTR을 낮출 것입니다..."

이후

"새로운 기능: 더 빠르고 간단" — 짧고 명확합니다. 부제목/설명: "주요 기능과 사용 예시를 모았습니다" (≤ 140자).

3) Telegram: '이전/이후'

이전

"[새로운 글] 모든 마케팅 채널을 위한 UTM 태그 완전 가이드, 목록, 예시, 체크리스트 및 목차 포함"

이후

"UTM: 완전 가이드" — "GA4, Google/Meta/VK용 템플릿, 일반적인 실수". 제목 ≤ 70자, 설명 ≤ 120–140자.

4) YouTube: '이전/이후'

이전

"30일 만에 SEO를 구축한 방법: 감사에서 구현, 분석 및 보고까지 모든 단계의 완전한 분석 [2025]" (105자)

이후

"30일 만에 SEO: 단계별 계획" — "감사, 구현, 보고. 실수와 생활 팁 (2025)". 제목 ≤ 60자, 설명의 처음 150–200자 — 주요 이점 및 CTA.

5) TikTok: '이전/이후'

이전

"많은 세부사항과 링크가 포함된 우리의 새로운 긴 설명을 시청하세요 — 끝까지 읽으세요!"

이후

"트래픽을 먹어버리는 3가지 UTM 실수" — "소스/미디엄, 인코딩 및 리디렉션을 확인하세요. 완전 가이드 — 링크를 통해".

6) 미리보기 및 메타데이터 팁

  • VK/Telegram에서 올바른 미리보기를 위해 OG 메타 (title/description/image)를 설정하세요.
  • 보편적 가시성을 위해 title ≤ 60–70자, description ≤ 140–160자를 유지하세요.
  • 설명의 처음 1–2줄에 주요 이점을 배치하세요.
  • 게시 전에 미리보기를 확인하세요 (초안/미리보기 봇/공유를 통해).
