🎨 현대 CSS 기술

1. CSS Container Queries

화면 크기가 아닌 컨테이너 크기에 따라 요소를 스타일링할 수 있는 새로운 CSS 기능입니다. 이는 반응형 디자인에서의 혁명입니다!

🎨 Container Queries 예제

.card {
    container-type: inline-size;
}

@container (max-width: 400px) {
    .card h2 {
        font-size: 1.2rem;
    }
    
    .card p {
        font-size: 0.9rem;
    }
}

@container (min-width: 401px) {
    .card {
        display: grid;
        grid-template-columns: 1fr 2fr;
    }
}

2. CSS Grid Level 2

새로운 CSS Grid 기능에는 복잡한 중첩 그리드를 만들 수 있는 하위 그리드(subgrid)가 포함됩니다.

3. CSS Houdini

개발자가 브라우저의 CSS 엔진에 직접 접근하여 사용자 정의 속성 및 애니메이션을 만들 수 있게 하는 API 세트입니다.

🔧 새로운 JavaScript 기능 예제

// 배열 그룹화 (ES2024)
const users = [
    { name: '앨리스', age: 25, city: '모스크바' },
    { name: '밥', age: 30, city: '상트페테르부르크' },
    { name: '찰리', age: 25, city: '모스크바' }
];

const groupedByAge = users.groupBy(user => user.age);
// 결과: { 25: [...], 30: [...] }

// 파이프라인 연산자 (제안)
const result = users
    |> filter(user => user.age > 25)
    |> map(user => user.name)
    |> join(', ');

// 패턴 매칭 (제안)
const response = match(fetch('/api/user')) {
    { status: 200, data } => data,
    { status: 404 } => '사용자를 찾을 수 없습니다',
    { status: 500 } => '서버 오류',
    _ => '알 수 없는 오류'
};

🌐 새로운 웹 표준

1. 웹 컴포넌트

모든 현대 브라우저에서 작동하는 재사용 가능한 구성 요소를 만들기 위한 표준입니다.

🔧 웹 컴포넌트 예제

class UserCard extends HTMLElement {
    constructor() {
        super();
        this.attachShadow({ mode: 'open' });
    }
    
    connectedCallback() {
        const name = this.getAttribute('name');
        const email = this.getAttribute('email');
        
        this.shadowRoot.innerHTML = `
            <style>
                .card {
                    border: 1px solid #ccc;
                    padding: 1rem;
                    border-radius: 8px;
                }
            </style>
            <div class="card">
                <h3>${name}</h3>
                <p>${email}</p>
            </div>
        `;
    }
}

customElements.define('user-card', UserCard);

// 사용법:
// <user-card name="홍길동" email="gildong@example.com"></user-card>
