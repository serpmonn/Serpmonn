What is Serpmonn.ru?

Serpmonn is a practical, everyday web toolkit: authentication and profiles, RSS news, browser games with leaderboards, and Telegram bots (gaming, confirmations, memes). Security is a priority: CSRF endpoints and rate limiting are implemented across core services. The goal of Serpmonn.ru is to bring all of this together in one convenient interface.

The project is evolving in the open: UX is being improved for mobile and desktop, new tools are added, accessibility is expanded (large text, high contrast, reduced motion). Plans include more HTML5 games, a better search experience, integrations, and a full profile system.

Security

The project is steadily hardening: PASETO tokens are used, CSRF endpoints implemented, rate limiting added, and transitive dependencies updated. Static analysis (CodeQL) and regular dependency updates are in place.

Technologies

  • Backend: Node.js, Express, MySQL, PASETO, PM2.
  • Frontend: HTML, CSS, JavaScript, PWA.
  • Infrastructure: Nginx, CI/CD, CodeQL.

Roadmap
