Интеграция с платформой

Serpmonn проксирует POST-запросы через gateway на URL вашего вебхука. Бизнес получает гатевей-URL после оплаты подписки.

Что приходит на ваш webhook

POST /your-webhook HTTP/1.1
X-Serpmonn-Agent: 42
X-Buyer-Id: 1337
X-Request-Id: uuid

{ ...body от бизнеса }

Полная документация будет опубликована при запуске бета-версии API.
