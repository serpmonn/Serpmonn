Интеграция с платформой

Serpmonn отправляет POST-запросы на URL вашего вебхука при каждом событии (сообщение пользователя, подключение, отключение).

Формат входящего запроса

POST /your-webhook HTTP/1.1
Content-Type: application/json
X-Serpmonn-Token: <your_secret_token>

{
  "event": "message",
  "user_id": "u_123456",
  "agent_id": "ag_789",
  "text": "Привет! Как заказать?",
  "ts": 1749638400
}

Формат ответа

HTTP/1.1 200 OK
Content-Type: application/json

{
  "reply": "Привет! Для заказа нажмите...",
  "actions": []  // опционально
}

Полная документация и SDK будут опубликованы при запуске бета-версии API (июль 2026).
