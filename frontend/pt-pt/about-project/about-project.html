O que é o Serpmonn.ru?

O Serpmonn é uma plataforma abrangente que se foca em ferramentas práticas: autenticação e perfis, notícias RSS, jogos de navegador com tabelas de líderes e bots do Telegram (jogos, verificação, memes). É dada especial atenção à segurança: foram adicionados endpoints CSRF e limitação de taxa aos serviços principais. O objetivo do Serpmonn.ru é combinar isto numa interface conveniente para tarefas diárias.

O projeto é desenvolvido abertamente: o UX em dispositivos móveis e desktop é gradualmente melhorado, são adicionadas ferramentas, a acessibilidade é expandida (fonte grande, alto contraste, menos animações). Os planos incluem mais jogos HTML5, pesquisa melhorada, integrações e um sistema de perfil completo.

Segurança

O projeto está a ser gradualmente fortalecido: são usados tokens PASETO, endpoints CSRF são implementados, limitação de taxa é adicionada, dependências transitivas são atualizadas. Análise estática (CodeQL) e atualizações regulares de dependências são realizadas.

Tecnologias

  • Backend: Node.js, Express, MySQL, PASETO, PM2.
  • Frontend: HTML, CSS, JavaScript, PWA.
  • Infraestrutura: Nginx, CI/CD, CodeQL.

Planos
