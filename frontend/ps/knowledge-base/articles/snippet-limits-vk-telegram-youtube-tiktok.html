د حروفو محدودیتونه او snippet پرې کول (VK / Telegram / YouTube / TikTok)

ولې دا ولولئ: ترڅو ډاډه اوسئ چې سرلیکونه او تشریحات پرې نه شي او اعظمي CTR وړاندې کړي. لاندې — د تصحیح لپاره د محدودیتونو لنډیز جدول او بصري 'مخکې/وروسته' مثالونه.

1) د محدودیتونو او وړاندیزونو لنډیز جدول

یادونه: پلیټفارمونه ډیری وخت UI او پرې کول بدلوي. موږ ازمایل شوي وړاندیز شوي اوږدوالۍ او پیژندل شوي سخت محدودیتونه وړاندې کوو.

پلیټفارم سرلیک (وړاندیز/محدودیت) تشریح/متن (وړاندیز/محدودیت) مخکتنه/پرې کول
VK ≤ ۶۰–۷۰ / — ≤ ۱۴۰–۱۶۰ / تر ۴۰۹۶ حروفو پورې په پوسټ کې د لینک snippet سرلیک ~۷۰، تشریح ~۱۴۰+ پرې کوي
Telegram ≤ ۷۰ / — (د OG څخه اخیستل شوی) ≤ ۱۲۰–۱۶۰ / ۴۰۹۶ په پیغام کې د OG څخه مخکتنه: سرلیک ~۷۰، تشریح ~۱۲۰–۱۴۰
YouTube ≤ ۶۰ / محدودیت ۱۰۰ ≤ ۱۵۰–۲۰۰ (لومړي کرښې) / محدودیت ۵۰۰۰ Above‑the‑fold نږدې ۲–۳ کرښې ښیي

2) VK: 'مخکې/وروسته'

مخکې

"زموږ د نوي خاصیت په اړه ډیر تفصيلي او ډیر اوږد سرلیک چې په ټولو مخکتنو کې به په ناچارۍ سره پرې شي او CTR به کم کړي..."

وروسته

„نوی خاصیت: ډیر چټک او ساده” — لنډ او څرګند. فرعي سرلیک/تشریح: „موږ اصلي وړتیاوې او د کارولو مثالونه راټول کړي دي” (≤ ۱۴۰).

3) Telegram: 'مخکې/وروسته'

مخکې

"[نوی مقاله] د ټولو بازارموندنې چینلونو لپاره د UTM ټېګونو بشپړ لارښود، په لیستونو، مثالونو، چیکلیست او منښتلیک سره"

وروسته

„UTM: بشپړ لارښود” — „GA4، د Google/Meta/VK لپاره ټیمپلیټونه، عام تېروتنې”. سرلیک ≤ ۷۰، تشریح ≤ ۱۲۰–۱۴۰.

4) YouTube: 'مخکې/وروسته'

مخکې

"موږ څنګه په ۳۰ ورځو کې SEO جوړ کړ: د آډیټ څخه تر پلي کولو، تحلیل او راپور ورکولو پورې د ټولو ګامونو بشپړ وېش [۲۰۲۵]" (۱۰۵ حروف)

وروسته

„په ۳۰ ورځو کې SEO: ګام په ګام پلان” — „آډیټ، پلي کول، راپور. تېروتنې او د ژوند لارښوونې (۲۰۲۵)”. سرلیک ≤ ۶۰، د تشریح لومړي ۱۵۰–۲۰۰ حروف — اصلي ګټې او CTA.

5) TikTok: 'مخکې/وروسته'

مخکې

"زموږ د نوي اوږد تشریح سره د ډیرو تفصیلاتو او لینکونو سره وګورئ — په یاد ولرئ چې تر پایه یې ولولئ!"

وروسته

„د ټرافیک خوړونکي ۳ د UTM تېروتنې” — „source/medium، کوډونه او ریډایرېکټونه وګورئ. بشپړ لارښود — د لینک له لارې".

6) د مخکتنې او metadata لارښوونې

  • په VK/Telegram کې د سمې مخکتنې لپاره OG meta (title/description/image) تنظیم کړئ.
  • د نړیوالې لیدلو لپاره title ≤ ۶۰–۷۰, description ≤ ۱۴۰–۱۶۰ وساتئ.
  • د تشریح لومړي ۱–۲ کرښو کې اصلي ګټې ځای پرځای کړئ.
  • د خپرولو دمخه مخکتنه وګورئ (د مسودو/د مخکتنې بوټانو/شریکولو له لارې).
