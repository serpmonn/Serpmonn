پورټ فوروارډینګ: بشپړ لارښود 2025

د اکتوبر 29، 2025 • 15 دقیقې لوستل

پورټ فوروارډینګ د کورني شبکې ترتیب کولو کې د نوي پیل کوونکو لپاره د ډیرو غوښتنې وړ او په ورته وخت کې ترټولو ستونزمنو موضوعاتو څخه یوه ده. په دې لارښود کې، به مو�هر د هرې موخې لپاره پورټ فوروارډینګ څرنګه په سمه توګه ترتیب کړو په تفصیل سره بحث وکړو: له لوبو سرورونو څخه تر څارنې سیستمونو پورې.

🔍 پورټ فوروارډینګ څه شی دی؟

⚠️ پوهیدل مهم دی

پورټ فوروارډینګ ستاسو وسیلې ته د انټرنیټ څخه لاسرسی پرانیزي. تل د امنیتي اقداماتو پیروي وکړئ او غیر اړین پورټونه مه پرانیزئ!

🔧 د ترتیب لپاره چمتووالی

  1. د وسیلې د ساکن IP پتې ټاکل

    ستاسو وسیله (PC، سرور، کیمره) ته د تلپاتې محلي IP پتې اړتیا ده ترڅو روټر پوه شي چې ګرځندوی چېرته لارښوونه کړي.

    په Windows کې د IP پتې معلومولو څرنګوالی:

    ipconfig

    د خپل شبکې اتصال لپاره د "IPv4 Address" کرښه وپلټئ

    په Linux/macOS کې د IP پتې معلومولو څرنګوالی:

    ifconfig

    یا کاروئ: ip addr show

  2. ساکن IP ترتیب کړئ

    د IP پتې د بدلون مخنیوي لپاره، دا ساکن ترتیب کړئ:

    • په روټر کې: د DHCP د رزرو له لارې ساکن IP وټاکئ
    • په وسیله کې: په شبکه ترتیباتو کې لاسي ساکن IP ترتیب کړئ

  3. خپله بهرنۍ IP پته ومومئ

    دا هغه پته ده چې نور کاروونکي یې په انټرنیټ کې ګوري:

    د بهرنۍ IP معلومولو لارې:

    # د команډي لاین له لارې
curl ifconfig.me

# یا ویب پاڼې وګورئ:
# - whatismyipaddress.com
# - 2ip.ru
# - myip.com

  4. اړین پورټونه وټاکئ

    معلومه کړئ چې ستاسو غوښتنلیک یا خدمت کوم پورټونه کاروي. دا معلومات معمولا په اسنادو کې دي.

🚀 د ترتیب پړاو په پړاو لارښوونې

  1. د روټر کنټرول پینل کې ننوځئ

    براوزر پرانیزئ او د روټر IP پته دننه کړئ. معمولا دا دي:

    • 192.168.1.1 (TP-Link, D-Link)
    • 192.168.0.1 (Netgear, ځینې TP-Link)
    • 192.168.31.1 (Xiaomi)
    • 192.168.10.1 (Huawei)

    د کارن نوم / پاسورډ معمولا د روټر په سټیکر کې لیست شوي دي.

  2. د پورټ فوروارډینګ برخه ومومئ

    برخه کې ډیر نومونه کیدی شي:

    • Port Forwarding
    • Virtual Servers
    • پورټ فوروارډینګ
    • NAT
    • فایر وال

  3. د پورټ فوروارډینګ پیرامیټرونه ډک کړئ

    تاسو باید وټاکئ:

    • د خدمت نوم: کوم نوم (لکه "Minecraft سرور")
    • بهرنی پورټ: په روټر کې به خلاصیدونکی پورټ
    • دننه پورټ: ستاسو په وسیله کې پورټ
    • دننه IP: ستاسو د وسیلې ساکن IP
    • پروتوکول: TCP، UDP یا دواړه

  4. ترتیبات خوندي کړئ

    ټول ساحې ډک کړې وروسته، "خوندي کړئ" یا "پلي کړئ" فشار کړئ. روټر بیا پیل کیدی شي.

📡 د مشهورو روټرونو لپاره لارښوونې

🔄 TP-Link

  1. د کنټرول پینل کې ننوځئ (192.168.1.1)
  2. ورځئ: پرمختللې → NAT فوروارډینګ → مجازی سرورونه
  3. په "Add New" کلیک وکړئ
  4. ډک کړئ:
    • Service Type: له لیست څخه غوره کړئ یا خپل دننه کړئ
    • External Port: د بهر لاسرسي لپاره پورټ
    • Internal Port: په وسیله کې پورټ
    • IP Address: د وسیلې محلي IP
    • Protocol: TCP/UDP/BOTH
  5. خوندي کړئ او روټر بیا پیل کړئ

🛡️ ASUS

  1. په پینل کې ننوځئ (192.168.1.1)
  2. ورځئ: پرمختللې ترتیبات → WAN → مجازی سرور/پورټ فوروارډینګ
  3. "Enable Port Forwarding" فعال کړئ
  4. ساحې ډک کړئ:
    • Service Name: د خدمت نوم
    • Port Range: د پورټ ساحه (لکه 25565-25565)
    • Local IP: ستاسو د وسیلې IP
    • Local Port: ستاسو د وسیلې پورټ
    • Protocol: TCP/UDP/BOTH
  5. په "Add" او بیا "Apply" کلیک وکړئ

🔗 D-Link

  1. په پینل کې ننوځئ (192.168.0.1)
  2. ورځئ: پرمختللې → پورټ فوروارډینګ
  3. په "Add" کلیک وکړئ
  4. ډک کړئ:
    • Rule Name: د قاعده نوم
    • External Port: بهرنی پورټ
    • Internal Port: دننه پورټ
    • Internal IP: د وسیلې IP
    • Schedule: تل
    • Inbound Filter: ټول اجازه کړئ
  5. ترتیبات خوندي کړئ

📶 Zyxel Keenetic

  1. په پینل کې ننوځئ (my.keenetic.net یا 192.168.1.1)
  2. ورځئ: امنیت → پورټ فوروارډینګ
  3. په "قاعده اضافه کړئ" کلیک وکړئ
  4. ډک کړئ:
    • تفصیل: د خدمت نوم
    • انٹرفیس: انټرنیټ
    • پروتوکول: TCP/UDP
    • بهرنی پورټ: د بهر لاسرسي لپاره پورټ
    • د هدف IP: د وسیلې محلي IP
    • دننه پورټ: په وسیله کې پورټ
  5. بدلونونه خوندي کړئ

✅ د پورټ فوروارډینګ د کار تایید

  1. آنلاین خدمتونو کارول

    د خلاصو پورټونو د چک لپاره ویب پاڼې:

    • portchecker.co
    • canyouseeme.org

  2. د کمانډي لاین له لارې تایید

    له بل کمپیوټر څخه چک کړئ:

    # telnet یا nc (netcat) کاروئ 
telnet ستاسو-بهرنی-IP پورټ 
# یا nmap کارولو سره 
nmap -p پورټ ستاسو-بهرنی-IP

  3. د خدمت ازموینه

    له محلي شبکې بهر د بل وسیلې څخه (لکه د موبایل انټرنیټ له لارې) خپل خدمت سره د وصل کیدو هڅه وکړئ.

✅ د بریالي ترتیب نښې

  • پورټ آنلاین چکرونو کې "خلاص" په توګه ښکاره کیږي
  • تاسو کولی شئ له بهر څخه خدمت سره وصل شئ
  • خدمت پرله پسې پرله پورې کار کوي

🔧 د عامو ستونزو حل

🚫 پورټ لا هم تړل شوی

  • فایر وال چک کړئ: په Windows فایر وال یا بل فایر وال کې استثنا اضافه کړئ
  • روټر بیا پیل کړئ: ځینې وخت ترتیبات یوازې بیا پیل کولو وروسته پلي کیږي
  • IP پته چک کړئ: ډاډ ترلاسه کړئ چې وسیله سم ساکن IP لري
  • UPnP غیر فعال کړئ: ځینې وخت UPnP د لاسي پورټ ترتیباتو سره نښتي

⚠️ د وړاندې کوونکي ډاینامیک IP

که تاسو ډاینامیک IP لرئ، دا بدلیدلی شي. حلونه:

  • DDNS کاروئ: د No-IP، DuckDNS په څیر خدمتونه
  • IP چک کړئ: د اوسني IP په اړه معلومات په منظم ډول تازه کړئ
  • ساکن IP: له وړاندې کوونکي څخه ساکن IP امر کړئ (تادیه شوی)

🔒 دوه ګونی NAT

ستونزه هغه وخت پیښیږي چې تاسو په شبکه کې ډیر روټرونه لرئ:

  • بریج حالت: یو له روټرونو څخه په بریج حالت کې وټاکئ
  • DMZ: د دوهم روټر لپاره په اصلي روټر کې DMZ ترتیب کړئ
  • په دواړو کې فوروارډینګ: په دواړو روټرونو کې فوروارډینګ ترتیب کړئ

🛡️ امنیتي اقدامات

  4. د روټر لاګونو په کارولو سره اړیکې څارئ
  5. د پورټ فوروارډینګ د خوندي بدیل په توګه VPN په پام کې ونیسئ
