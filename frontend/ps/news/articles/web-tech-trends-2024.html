🎨 عصري CSS ټېکنالوژۍ

1. CSS Container Queries

یوه نوی CSS ځانتيا چې اجازه ورکوي چې تاسو عناصر د دوی د کانټينر په کچه باندې ډيزاين کړئ، نه د سکرين کچې پر بنسټ. دا په رسپانسو ډيزاين کې یوه انقلاب دی!

🎨 د Container Queries بېلګه

.card {
    container-type: inline-size;
}

@container (max-width: 400px) {
    .card h2 {
        font-size: 1.2rem;
    }
    
    .card p {
        font-size: 0.9rem;
    }
}

@container (min-width: 401px) {
    .card {
        display: grid;
        grid-template-columns: 1fr 2fr;
    }
}

🌐 نوي وېب معيارونه

1. وېب کامپوننټونه

د بيازيات کاريدونکو کامپوننټونو د جوړولو لپاره معيارونه چې په ټولو عصري براوزرونو کې کار کوي.

🔧 د وېب کامپوننټ بېلګه

class UserCard extends HTMLElement {
    constructor() {
        super();
        this.attachShadow({ mode: 'open' });
    }
    
    connectedCallback() {
        const name = this.getAttribute('name');
        const email = this.getAttribute('email');
        
        this.shadowRoot.innerHTML = `
            <style>
                .card {
                    border: 1px solid #ccc;
                    padding: 1rem;
                    border-radius: 8px;
                }
            </style>
            <div class="card">
                <h3>${name}</h3>
                <p>${email}</p>
            </div>
        `;
    }
}

customElements.define('user-card', UserCard);

// کارونه:
// <user-card name="احمد شاه" email="ahmad@example.com"></user-card>
