Wat is Serpmonn.ru?

Serpmonn is een uitgebreid platform dat zich richt op praktische tools: authenticatie en profielen, RSS-nieuws, browserspellen met scoreborden en Telegram-bots (gaming, verificatie, memes). Bijzondere aandacht wordt besteed aan beveiliging: CSRF-endpoints en rate limiting zijn toegevoegd aan de kernservices. Het doel van Serpmonn.ru is om dit te combineren in een handige interface voor dagelijkse taken.

Het project wordt open ontwikkeld: UX op mobiel en desktop wordt geleidelijk verbeterd, tools worden toegevoegd, toegankelijkheid wordt uitgebreid (groot lettertype, hoog contrast, minder animaties). Plannen omvatten meer HTML5-spellen, verbeterde zoekfunctie, integraties en een volwaardig profielensysteem.

Beveiliging

Het project wordt geleidelijk versterkt: PASETO-tokens worden gebruikt, CSRF-endpoints worden geïmplementeerd, rate limiting wordt toegevoegd, transitieve afhankelijkheden worden bijgewerkt. Statische analyse (CodeQL) en regelmatige dependency-updates worden uitgevoerd.

Technologieën

  • Backend: Node.js, Express, MySQL, PASETO, PM2.
  • Frontend: HTML, CSS, JavaScript, PWA.
  • Infrastructuur: Nginx, CI/CD, CodeQL.
