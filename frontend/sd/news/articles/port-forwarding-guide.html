پورٽ فارورڊنگ: مڪمل رهنماء 2025

آڪٽوبر 29، 2025 • 15 منٽن جي پڙهائي

پورٽ فارورڊنگ گهرجوي نيٽورڪ ترتيب ۾ نوان شروع ڪندڙن لاءِ سڀني کان وڌيڪ گهربل ۽ ساڳئي وقت سڀني کان مشڪل موضوعن مان هڪ آهي. هن رهنماء ۾، اسان تفصيل سان بحث ڪنداسين ته ڪنهن به مقصد لاءِ پورٽ فارورڊنگ کي صحيح طريقي سان ڪيئن ترتيب ڏجي: راند سرورن کان نگراني سسٽمن تائين.

🔍 پورٽ فارورڊنگ ڇا آهي؟

پورٽ فارورڊنگ هڪ ٽيڪنالاجي آهي جيڪا نيٽورڪ جي درخواستن کي روٽر جي ٻاهرين پورٽ کان توهان جي مقامي نيٽورڪ ۾ ڪنهن ڊوائيس جي خاص پورٽ ڏانهن موڙيندي آهي.

⚠️ سمجهڻ ضروري آهي

پورٽ فارورڊنگ توهان جي ڊوائيس تي انٽرنيٽ کان رسائي کوليو ٿو. هميشه حفاظتي قدمن تي عمل ڪريو ۽ غير ضروري پورٽ نه کوليو!

🔧 ترتيب لاءِ تيارڻي

  1. ڊوائيس جو اسٽيٽڪ IP پتو طئي ڪريو

    توهان جي ڊوائيس (PC، سرور، ڪئميرا) کي هڪ مستقل مقامي IP پتو گهربل آهي ته جيئن روٽر کي خبر هجي ته ٽريفڪ ڪٿي موڙي.

    ونڊوز ۾ IP پتو ڪيئن ڳولجي:

    ipconfig

    توهان جي نيٽورڪ ڪنيڪشن لاءِ "IPv4 Address" لائين ڳوليو

    Linux/macOS ۾ IP پتو ڪيئن ڳولجي:

    ifconfig

    يا استعمال ڪريو: ip addr show

  2. اسٽيٽڪ IP ترتيب ڏيو

    IP پتو جي تبديلين کان بچڻ لاءِ، انهي کي اسٽيٽڪ ترتيب ڏيو:

    • روٽر ۾: DHCP رزرو ذريعي اسٽيٽڪ IP مقرر ڪريو
    • ڊوائيس تي: نيٽورڪ ترتيبن ۾ دستي طور اسٽيٽڪ IP ترتيب ڏيو

  3. توهان جو ٻاهرين IP پتو ڄاڻيو

    اهو پتو آهي جيڪو ٻين استعمال ڪندڙن انٽرنيٽ تي ڏسون ٿا:

    ٻاهرين IP ڄاڻڻ جا طريقا:

    # ڪمانڊ لائن ذريعي
curl ifconfig.me

# يا ويب سائيٽون ڏسو:
# - whatismyipaddress.com
# - 2ip.ru
# - myip.com

  4. گهربل پورٽ طئي ڪريو

    ڄاڻيو ته توهان جو ايپليڪيشن يا سروس ڪهڙا پورٽ استعمال ڪري ٿو. اهو معلومات عام طور تي دستاويزن ۾ هوندو آهي.

🚀 مرحلا وار ترتيب جون هدايتون

  1. روٽر ڪنٽرول پينل ۾ لاگ ان ٿيو

    برائوزر کوليو ۽ روٽر جو IP پتو داخل ڪريو. عام طور تي اهو آهي:

    • 192.168.1.1 (TP-Link, D-Link)
    • 192.168.0.1 (Netgear، ڪجهه TP-Link)
    • 192.168.31.1 (Xiaomi)
    • 192.168.10.1 (Huawei)

    صارف نالو/پاسورڊ عام طور تي روٽر تي اسٽڪر تي درج ٿيل آهن.

  2. پورٽ فارورڊنگ حصو ڳوليو

    حصو جا مختلف نالا ٿي سگهن ٿا:

    • Port Forwarding
    • Virtual Servers
    • پورٽ فارورڊنگ
    • NAT
    • فائر وال

  3. پورٽ فارورڊنگ پاراميٽر ڀريو

    توهان کي بيان ڪرڻ گهربل آهي:

    • سروس جو نالو: ڪو به نالو (مثال طور "Minecraft سرور")
    • ٻاهرين پورٽ: روٽر تي کليل ٿيندڙ پورٽ
    • اندروني پورٽ: توهان جي ڊوائيس تي پورٽ
    • اندروني IP: توهان جي ڊوائيس جو اسٽيٽڪ IP
    • پروٽوڪول: TCP، UDP يا ٻئي

  4. ترتيب محفوظ ڪريو

    سڀني فيلڊن کي ڀرڻ کان پوء، "محفوظ ڪريو" يا "لاڳو ڪريو" دٻايو. روٽر ٻيهر شروع ٿي سگهي ٿو.

📡 مقبول روٽرن لاءِ هدايتون

🔄 TP-Link

  1. ڪنٽرول پينل ۾ لاگ ان ٿيو (192.168.1.1)
  2. وڃو: ايڊوانسڊ → NAT فارورڊنگ → ورچوئل سرورز
  3. "Add New" تي ڪلڪ ڪريو
  4. ڀريو:
    • Service Type: فهرست مان چونڊيو يا پنهنجو داخل ڪريو
    • External Port: ٻاهرين رسائي لاءِ پورٽ
    • Internal Port: ڊوائيس تي پورٽ
    • IP Address: ڊوائيس جو مقامي IP
    • Protocol: TCP/UDP/BOTH
  5. محفوظ ڪريو ۽ روٽر ٻيهر شروع ڪريو

🛡️ ASUS

  1. پينل ۾ لاگ ان ٿيو (192.168.1.1)
  2. وڃو: ايڊوانسڊ سيٽنگز → WAN → ورچوئل سرور/پورٽ فارورڊنگ
  3. "Enable Port Forwarding" فعال ڪريو
  4. فيڊز ڀريو:
    • Service Name: سروس جو نالو
    • Port Range: پورٽ جي حد (مثال طور 25565-25565)
    • Local IP: توهان جي ڊوائيس جو IP
    • Local Port: توهان جي ڊوائيس جو پورٽ
    • Protocol: TCP/UDP/BOTH
  5. "Add" ۽ پوءِ "Apply" تي ڪلڪ ڪريو

🔗 D-Link

  1. پينل ۾ لاگ ان ٿيو (192.168.0.1)
  2. وڃو: ايڊوانسڊ → پورٽ فارورڊنگ
  3. "Add" تي ڪلڪ ڪريو
  4. ڀريو:
    • Rule Name: قاعدي جو نالو
    • External Port: ٻاهرين پورٽ
    • Internal Port: اندروني پورٽ
    • Internal IP: ڊوائيس جو IP
    • Schedule: هميشه
    • Inbound Filter: سڀني کي اجازت ڏيو
  5. ترتيب محفوظ ڪريو

📶 Zyxel Keenetic

  1. پينل ۾ لاگ ان ٿيو (my.keenetic.net يا 192.168.1.1)
  2. وڃو: سيڪيورٽي → پورٽ فارورڊنگ
  3. "قاعدو شامل ڪريو" تي ڪلڪ ڪريو
  4. ڀريو:
    • وضاحت: سروس جو نالو
    • انٽرفيس: انٽرنيٽ
    • پروٽوڪول: TCP/UDP
    • ٻاهرين پورٽ: ٻاهرين رسائي لاءِ پورٽ
    • مقصد IP: ڊوائيس جو مقامي IP
    • اندروني پورٽ: ڊوائيس تي پورٽ
  5. تبديلين کي محفوظ ڪريو

✅ پورٽ فارورڊنگ جي ڪم جي تصديق

  1. آنلائن سروسز استعمال ڪريو

    کليل پورٽن جي چڪاس لاءِ ويب سائيٽون:

    • portchecker.co
    • canyouseeme.org

  2. ڪمانڊ لائن ذريعي تصديق

    ٻي ڪمپيوٽر کان چيڪ ڪريو:

    # telnet يا nc (netcat) استعمال ڪريو 
telnet توهان-جو-ٻاهرين-IP پورٽ 
# يا nmap استعمال ڪندي 
nmap -p پورٽ توهان-جو-ٻاهرين-IP

  3. سروس جي جانچ

    پنهنجي مقامي نيٽورڪ کان ٻاهر ٻي ڊوائيس مان (مثال طور موبائيل انٽرنيٽ ذريعي) پنهنجي سروس سان ڳنڍڻ جي ڪوشش ڪريو.

✅ ڪامياب ترتيب جا نشان

  • پورٽ آنلائن چيڪرز ۾ "کليل" طور ڏيکاري ٿو
  • توهان ٻاهرين کان سروس سان ڳنڍڻ سگهو ٿا
  • سروس بغير روڪ ٿام مستقل ڪم ڪري ٿي

🔧 عام مسئلن جو حل

🚫 پورٽ اڃا به بند آهي

  • فائر وال چيڪ ڪريو: ونڊوز فائر وال يا ٻي فائر وال ۾ استثناٰ شامل ڪريو
  • روٽر ٻيهر شروع ڪريو: ڪڏهن ڪڏهن ترتيب صرف ٻيهر شروع ڪرڻ کان پوءِ لاڳو ٿيندي آهي
  • IP پتو چيڪ ڪريو: يقيني بڻايو ته ڊوائيس صحيح اسٽيٽڪ IP رکي ٿي
  • UPnP غير فعال ڪريو: ڪڏهن ڪڏهن UPnP دستي پورٽ ترتيبن سان متصادم ٿي سگهي ٿو

⚠️ فراهم ڪندڙ جو ڊائنامڪ IP

جيڪڏهن توهان وٽ ڊائنامڪ IP آهي، اهو تبديل ٿي سگهي ٿو. حل:

  • DDNS استعمال ڪريو: No-IP، DuckDNS جهڙيون سروسز
  • IP چيڪ ڪريو: موجوده IP بابت معلومات باقاعده تازه ڪريو
  • اسٽيٽڪ IP: فراهم ڪندڙ کان اسٽيٽڪ IP آرڊر ڪريو (ادا ڪيل)

🔒 ڊبل NAT

مسئلو اهڙو ٿو ٿئي جڏهن توهان وٽ نيٽورڪ ۾ ڪيترائي روٽر هجن:

  • برج موڊ: روٽرن مان هڪ کي برج موڊ ۾ سيٽ ڪريو
  • DMZ: ٻئي روٽر لاءِ مکيه روٽر تي DMZ ترتيب ڏيو
  • ٻنهي تي فارورڊنگ: ٻنهي روٽرن تي فارورڊنگ ترتيب ڏيو

🛡️ حفاظتي قدم

🔐 اهم قاعدا

  1. صرف ضروري پورٽ کوليو - هر کليل پورٽ حملو جي سطحي وڌائي ٿو
  2. ٻاهرين رسائي واري سڀني سروسز تي پيچيدو پاسورڊ استعمال ڪريو
  3. سافٽوير باقاعده تازه ڪريو - ڪمزورين کي ختم ڪريو
  4. روٽر لاگز استعمال ڪندي ڪنيڪشنز جي نگراني ڪريو
  5. پورٽ فارورڊنگ جي هڪ محفوظ متبادل طور VPN تي غور ڪريو

🎯 بهترين عمل

  • اسڪينرن جي توجہ گهٽائڻ لاءِ غير معياري پورٽ استعمال ڪريو
  • شيڊول موجب پورٽن جي خودڪار غير فعال ٿيڻ جو ترتيب ڏيو
  • مشڪوه سرگرمي لاءِ باقاعده لاگز چيڪ ڪريو
  • مشڪوه IPs کي بلاڪ ڪرڻ لاءِ fail2ban يا ساڳيو استعمال ڪريو
