IndexNow: عملي عمل ۽ صلاحون

1) IndexNow ڇا آهي

IndexNow هڪ پروٽوڪول آهي جيڪو توهان کي سرچ انجنن کي URL تبديلين (تخليق، اپڊيٽ، ختم ڪرڻ) بابت فوري طور تي اطلاع ڏيڻ جي اجازت ڏئي ٿو. Bing، Yandex، Seznam ۽ ٻين خدمتن طرفان سپورٽ ڪيل. فائدو گهٽ ۾ گهٽ انڊيڪسنگ تاخير ۽ گهٽ ڪرول شور آهي.

2) ڪنجي جو سيٽ اپ ۽ بنيادي پنگ

  1. هڪ ڪنجي (GUID) پيدا ڪريو ۽ YOUR_KEY.txt فائيل سائيٽ جي جڙ (root) ۾ (/.well-known/ يا ڊومين جڙ) ۾ ڪنجي جي تار سان رکو.
  2. URL ۽ ڪنجي سان اينڊپوائنٽ تي پنگ موڪليو. مثال (هڪ واحد URL):
POST https://api.indexnow.org/indexnow Content-Type: application/json { "host": "www.serpmonn.ru", "key": "YOUR_KEY", "urlList": [ "https://www.serpmonn.ru/frontend/news/articles/utm-complete-guide.html" ] }
بيچن لاءِ، هڪ درخواست ۾ 10،000 تائين URL بيان ڪريو يا تبديلي لاگ فائيل جي URL سان فائيل فيڊ استعمال ڪريو.

3) صرف-تبديل ٿيل موڊ

خيال اهو آهي ته صرف انهن فائيلن تي پنگ موڪليو جيڪي آخري commit/ڊپلائيمينٽ ۾ اصل ۾ تبديل ٿيل آهن. اهو شور ۽ ڪوٽا جي استعمال کي گهٽائيندو آهي.

  • VCS (git diff) يا بلڊ آرٽيفيڪٽس مان تبديل ٿيل رستن جي فهرست حاصل ڪريو.
  • عوامي رستن جي مطابق فلٽر ڪريو (node_modules، .git، ذاتي فولڊرن کان سواءِ).
  • غير متعلق URL موڪلڻ کان بچڻ لاءِ ڊومين/ٻولين جي لحاظ کان گروپ ڪريو.
پيداوار ۾، فال بيڪ رکڻ مفيد آهي: جيڪڏهن صرف-تبديل ٿيل خالي آهي (مثال طور ترتيب لاءِ هيٽ فڪس)، اهم سائيٽ ميپن تي پنگ موڪليو.

4) IndexNow بمقابلو sitemap ping

  • رفتار: IndexNow تيزي سان اپڊيٽس پهيڙيندو آهي، ڪرول چڪر لاءِ انتظار نه ڪندو؛ sitemap ping سائيٽ ميپ جي موجودگي جي اشارو ڏئي ٿو پر ڪنهن مخصوص URL جي تبديلين جو نه.
  • شور: IndexNow - تبديل ٿيل URLن تي نشانو؛ sitemap - سڄي نقشي جي اسڪين (ڪرول بجيٽ لاءِ وڌيڪ مھانگو).
  • قابلي اعتماد: سائيٽ ميپس بنيادي طور تي رهندا آهن؛ IndexNow مڪمل ڪري ٿو ۽ تيز ڪري ٿو. ٻئي استعمال ڪرڻ بهتر آهي.

5) عام غلطيون

  • ڪنجي عوامي URL ذريعي قابل رسائي ناهي يا key پیرامیٽر سان ملندڙ ناهي.
  • ذاتي/404 صفحن تي پنگ، ڪينونيڪلائيزيشن کان سواءِ پیرامیٽر.
  • ساڳين URLن جا تمام گهڻا وراڻا (واراڻا محدود ڪريو).
  • ڪيترين ئي ٻولين کان نظر انداز ڪرڻ (لاڳاپيل مقامي ٿيل ورزن کي پنگ ڪرڻ ضروري آهي).

6) ڪثرت سان پڇيا ويندڙ سوال ۽ چيڪ لسٽ

هڪ ئي وقت ۾ ڪيترا URL موڪليان؟

هڪ بيچ ۾ 10،000 تائين - قسم/ٻولي جي لحاظ کان بهترين گروپنگ.

ڇا ختم ڪرڻ تي پنگ موڪليان؟

ها - ختم ٿيل صفحن جا URL ڊي انڊيڪسنگ تيز ڪرڻ لاءِ پهچايو.

ڪيئن تصديق ڪجي ته ڪم ڪري ٿو؟

API جواب رڪارڊ (200/202)، سرور رڪارڊ ۾ ڪرول چوٽيون، نوان URL تيزي سان ظاهر ٿيڻ.

  • ڪنجي رکيل آهي ۽ HTTPS ذريعي قابل رسائي آهي.
  • صرف-تبديل ٿيل رستا CI/CD کان گڏ ڪريو.
  • ڊيڊپليڪيشن ۽ وراڻن جي پابندي.
  • خالي بيچ تي sitemap ping ڏانهن فال بيڪ.
