Introduction

The serpmonn.ru website values user privacy and takes all necessary measures to protect personal data. This privacy policy describes how and for what purposes user data is collected, used, stored, and protected. By using this website, the user agrees to the terms set forth in this policy.

1. Collected Data

The website may collect the following types of data:

  • Personal data: name, email address, phone number.
  • Technical data: IP address, browser type, operating system, website visit information (including pages visited, date and time of visit).
  • User-provided data: information provided by the user when filling out forms on the website, including comments, feedback, requests, and other feedback.

2. Data Usage

Data is used for the following purposes:

  • To provide and improve services.
  • To process requests and feedback.
  • For analysis and research to improve website performance and ensure security.
  • To protect against unauthorized access and prevent attacks on the website.
  • To comply with obligations to users.

3. Data Storage

Data is stored for the time necessary to achieve the specified purposes:

  • Personal data (name, email, phone) are stored until deletion is requested or services are terminated.
  • Technical data (IP addresses, browser data) are stored for no more than 30 days.

4. Data Sharing with Third Parties

Users' personal data is not shared with third parties.

5. Data Protection

All necessary measures are taken to protect data from unauthorized access, use, or disclosure. Personal data is protected using modern encryption technologies and other security methods. However, despite the measures taken, absolute security cannot be guaranteed.

6. User Rights

The user has the right to:

  • Request access to stored personal data.
  • Request correction or deletion of data.
  • Object to data processing in certain cases.
  • Withdraw consent to the processing of personal data.

7. Contact Information

For questions related to the privacy policy, data processing, or exercising rights, please contact: privacy@serpmonn.ru.

Terms of service and disclaimer are available in the Disclaimer section.

8. Privacy Policy Changes

All changes to the privacy policy are published on this page. It is recommended to regularly check for updated information.
