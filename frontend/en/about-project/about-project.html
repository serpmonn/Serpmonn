What is Serpmonn.ru?

Serpmonn is a comprehensive platform focusing on practical tools: authentication and profiles, RSS news, browser games with leaderboards and Telegram bots (gaming, verification, memes). Special attention is paid to security: CSRF endpoints and rate limiting have been added to core services. The goal of Serpmonn.ru is to combine these in a convenient interface for everyday tasks.

The project is developed openly: UX on mobile and desktop is gradually improved, tools are added, accessibility is expanded (large font, high contrast, fewer animations). Plans include more HTML5 games, improved search, integrations and a full-fledged profile system.

Project Features

  • Authentication and Profiles — Secure authorization (PASETO), profile management.
  • Information Analysis Tools — 7 services for data collection and processing, RSS news.
  • Platform Games — 19 browser and PC games with leaderboards.
  • Telegram Bots — Gaming bot, verification bot and meme bot.
  • PWA — Progressive web app support.
  • Security — CSRF endpoints and rate limiting in core services.

Who Can Benefit

  • Users — convenient access to news, games and services in one place.
  • Developers — open repository with clear structure, possibility to contribute.
  • Bot and Mini-Service Authors — platform for integration and testing ideas.

What Already Works

  • Authentication and profile system.
  • News from RSS sources (9 articles).
  • Games (19 games, including leaderboards).

  • CSRF endpoints and rate limiting.

Security

The project is gradually being strengthened: PASETO tokens are used, CSRF endpoints are implemented, rate limiting is added, transitive dependencies are updated. Static analysis (CodeQL) and regular dependency updates are performed.

Technologies

  • Backend: Node.js, Express, MySQL, PASETO, PM2.
  • Frontend: HTML, CSS, JavaScript, PWA.
  • Infrastructure: Nginx, CI/CD, CodeQL.

Plans

  • Developing bots and integrations, optimizing performance.
