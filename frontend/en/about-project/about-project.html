What is Serpmonn.ru?

Serpmonn is a convenient set of tools and entertainment with ad-free and tracking-free search. Everything in one place: search, news, games, tools and promo codes!

Who can benefit from this

  • For users — convenient access to news, games and services in one place.
  • For developers — open repository with clear structure and possibility of making changes.

What already works

  • Registration system and personal profiles
  • Fresh news from RSS sources
  • Games (19 games, including leaderboards)
  • 7 tools for work

Plans

  • Search improvement — add 'smart'/AI search option (summaries, comparison tables, direct answers without many links)
  • More tools — add 3–5 new ones (e.g., QR code generator, real-time currency converter, simple task planner or budget calculator).
  • PWA/TWA improvement — make Serpmonn Beta more stable, improve offline access.
  • Community and feedback — add a simple ideas/voting form on the site, possibly a Telegram channel or chat for discussions and voting on new features.
  • Expanding the set of games and profile features — add new mini-games, improve leaderboards, add achievements, avatars and statistics in the profile.
