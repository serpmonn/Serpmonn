How to play "Falling blocks"?

  • Controls: Use arrow keys or touch to move the red square left and right
  • Goal: Dodge the falling objects
  • Scoring: You receive 1 point for each avoided object
  • Misses: If an object hits you it counts as a miss
  • Game over: The game ends after 10 hits
  • Leaderboard: After the game you can view your results on the leaderboard
  • Pause: Use the "Pause" button to temporarily stop the game
