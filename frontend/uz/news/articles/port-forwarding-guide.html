‚ö†Ô∏è Tushunish muhim

Port yo'naltirish qurilmangizga internet orqali kirishni ochadi. Har doim xavfsizlik choralariga rioya qiling va keraksiz portlarni ochmang!

üîß Sozlashga tayyorgarlik

  1. Qurilmaning statsik IP manzilini aniqlang

    Sizning qurilmangiz (PC, server, kamera) router trafikni qayerga yo'naltirishni bilishi uchun doimiy mahalliy IP manziliga ega bo'lishi kerak.

    Windows'da IP manzilini qanday topish kerak:

    ipconfig

    Tarmoq ulashingiz uchun "IPv4 Address" qatorini qidiring

    Linux/macOS'da IP manzilini qanday topish kerak:

    ifconfig

    Yoki ishlating: ip addr show

  2. Statsik IP sozlang

    IP manzilining o'zgarishining oldini olish uchun uni statik ravishda sozlang:

    • Router'da: DHCP rezervatsiyasi orqali statsik IP belgilang
    • Qurilmada: Tarmoq sozlamalarida qo'lda statsik IP sozlang

  3. Tashqi IP manzilingizni bilib oling

    Bu boshqa foydalanuvchilar internetda ko'radigan manzil:

    Tashqi IP ni bilish usullari:

    # Buyruq qatori orqali
curl ifconfig.me

# Yoki veb-saytlarni tashrif qiling:
# - whatismyipaddress.com
# - 2ip.ru
# - myip.com

  4. Kerakli portlarni aniqlang

    Ilovangiz yoki xizmat qaysi portlardan foydalanishini aniqlang. Bu ma'lumot odatda hujjatlarda mavjud.

üöÄ Bosqichma-bosqich sozlash ko'rsatmalari

  1. Router boshqaruv paneliga kiring

    Brauzeringizni oching va router IP manzilini kiriting. Odatda bu:

    • 192.168.1.1 (TP-Link, D-Link)
    • 192.168.0.1 (Netgear, ba'zi TP-Link)
    • 192.168.31.1 (Xiaomi)
    • 192.168.10.1 (Huawei)

    Foydalanuvchi nomi/parol odatda router'ning yorlig'ida ko'rsatilgan.

  2. Port yo'naltirish bo'limini toping

    Bo'lim turli nomlarga ega bo'lishi mumkin:

    • Port Forwarding
    • Virtual Servers
    • Port Yo'naltirish
    • NAT
    • Firewall

  3. Port yo'naltirish parametrlarini to'ldiring

    Quyidagilarni ko'rsatishingiz kerak:

    • Xizmat Nomi: Har qanday nom (masalan, "Minecraft Server")
    • Tashqi Port: Router'da ochiladigan port
    • Ichki Port: Qurilmangizdagi port
    • Ichki IP: Qurilmangizning statsik IP'si
    • Protokol: TCP, UDP yoki Ikkalasi

  4. Sozlamalarni saqlang

    Barcha maydonlarni to'ldirgandan so'ng, "Saqlash" yoki "Qo'llash" tugmasini bosing. Router qayta ishga tushishi mumkin.

üì° Mashhur Router'lar uchun ko'rsatmalar

üîÑ TP-Link

  1. Boshqaruv paneliga kiring (192.168.1.1)
  2. O'ting: Advanced ‚Üí NAT Forwarding ‚Üí Virtual Servers
  3. "Add New" tugmasini bosing
  4. To'ldiring:
    • Service Type: Ro'yxatdan tanlang yoki o'zingiz kiring
    • External Port: Tashqaridan kirish uchun port
    • Internal Port: Qurilmadagi port
    • IP Address: Qurilmaning mahalliy IP'si
    • Protocol: TCP/UDP/BOTH
  5. Saqlang va router'ni qayta ishga tushiring

üõ°Ô∏è ASUS

  1. Panelga kiring (192.168.1.1)
  2. O'ting: Advanced Settings ‚Üí WAN ‚Üí Virtual Server/Port Forwarding
  3. "Enable Port Forwarding" ni yoqing
  4. Maydonlarni to'ldiring:
    • Service Name: Xizmat nomi
    • Port Range: Port oralig'i (masalan, 25565-25565)
    • Local IP: Qurilmangizning IP'si
    • Local Port: Qurilmangizning porti
    • Protocol: TCP/UDP/BOTH
  5. "Add" keyin "Apply" tugmasini bosing

üîó D-Link

  1. Panelga kiring (192.168.0.1)
  2. O'ting: Advanced ‚Üí Port Forwarding
  3. "Add" tugmasini bosing
  4. To'ldiring:
    • Rule Name: Qoida nomi
    • External Port: Tashqi port
    • Internal Port: Ichki port
    • Internal IP: Qurilmaning IP'si
    • Schedule: Always
    • Inbound Filter: Allow All
  5. Sozlamalarni saqlang

üì∂ Zyxel Keenetic

  1. Panelga kiring (my.keenetic.net yoki 192.168.1.1)
  2. O'ting: Xavfsizlik ‚Üí Port Yo'naltirish
  3. "Qoida qo'shish" tugmasini bosing
  4. To'ldiring:
    • Tavsif: Xizmat nomi
    • Interfeys: Internet
    • Protokol: TCP/UDP
    • Tashqi port: Tashqaridan kirish uchun port
    • Maqsad IP: Qurilmaning mahalliy IP'si
    • Ichki port: Qurilmadagi port
  5. O'zgarishlarni saqlang

‚úÖ Port Yo'naltirish ishlashini tekshirish

  1. Onlayn xizmatlardan foydalaning

    Ochiq portlarni tekshirish uchun veb-saytlar:

  3. Xizmatni sinash

    Mahalliy tarmog'ingizdan tashqarida boshqa qurilmadan (masalan, mobil internet orqali) xizmatingizga ulanishga harakat qiling.

üîß Umumiy muammolarni hal qilish

üö´ Port hali ham yopiq

  • Firewall'ni tekshiring: Windows Firewall yoki boshqa firewall'da istisno qo'shing
  • Router'ni qayta ishga tushiring: Ba'zan sozlamalar faqat qayta ishga tushirilgandan keyin qo'llaniladi
  • IP manzilini tekshiring: Qurilma to'g'ri statsik IP'ga ega ekanligiga ishonch hosil qiling
  • UPnP ni o'chiring: Ba'zan UPnP qo'lda port sozlamalari bilan to'qnashadi

‚ö†Ô∏è Provayderning dinamik IP'si

Agar sizda dinamik IP bo'lsa, u o'zgarishi mumkin. Yechimlar:

  • DDNS dan foydalaning: No-IP, DuckDNS kabi xizmatlar
  • IP ni tekshiring: Joriy IP haqida ma'lumotlarni muntazam yangilang
  • Statsik IP: Provayderdan statsik IP buyurtma qiling (pullik)

üîí Ikkilamchi NAT

  • Ko'prik rejimi: Router'lardan birini ko'prik rejimiga o'tkazing
  • DMZ: Ikkinchi router uchun asosiy router'da DMZ sozlang
  • Ikkalasida yo'naltirish: Ikkala router'da ham yo'naltirishni sozlang
