  2. HTML Tuzilmasini Yaratish

    Asosiy HTML tuzilmasi bilan boshlang. HTML sizning veb-saytingizning skeletidir.

    Asosiy HTML shabloni:

    <!DOCTYPE html>
<html lang="uz">

    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mening Birinchi Veb-saytim</title>

<body>
    <h1>Xush kelibsiz!</h1>
    <p>Bu mening birinchi veb-saytim.</p>
</body>
</html>
