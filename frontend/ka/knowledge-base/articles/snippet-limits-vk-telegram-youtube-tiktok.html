1) ლიმიტების და რეკომენდაციების შეჯამებული ცხრილი

შენიშვნა: პლატფორმები ხშირად ცვლის UI-ს და შეკვეცას. ჩვენ ვაწვდით გამოცდილ რეკომენდებულ სიგრძეებს და ცნობილ მკაცრ ლიმიტებს.

პლატფორმა სათაური (რეკომ./ლიმიტი) აღწერილობა/ტექსტი (რეკომ./ლიმიტი) წინასწარი გადახედვა/შეკვეცა
VK ≤ 60–70 / — ≤ 140–160 / პოსტში 4096 სიმბოლომდე ლინკ snippet შეკვეცავს სათაურს ~70, აღწერილობას ~140+
Telegram ≤ 70 / — (OG-დან აღებული) ≤ 120–160 / შეტყობინებაში 4096 წინასწარი გადახედვა OG-დან: სათაური ~70, აღწერილობა ~120–140
YouTube ≤ 60 / ლიმიტი 100 ≤ 150–200 (პირველი სტრიქონები) / ლიმიტი 5000 Above‑the‑fold აჩვენებს ~2–3 სტრიქონს
TikTok ≤ 60–80 / — ≤ 80–150 (ხილული ნაწილი) / ლიმიტი ~2200-მდე წარწერა შეკვეცილია, «მეტი» ~80–150-ის შემდეგ

2) VK: «ადრე/შემდეგ»

ადრე

«ჩვენი ახალი ფუნქციის შესახებ წარმოუდგენლად დეტალური და ძალიან გრძელი სათაური, რომელიც გარდაუვლად შეიკვეცება ყველა წინასწარ გადახედვაში და შეამცირებს CTR-ს…»

შემდეგ

«ახალი ფუნქცია: უფრო სწრაფი და მარტივი» — მოკლე და ნათელი. ქვესათაური/აღწერილობა: «ჩვენ შევკრიბეთ ძირითადი შესაძლებლობები და გამოყენების მაგალითები» (≤ 140).

3) Telegram: «ადრე/შემდეგ»

ადრე

«[ახალი სტატია] UTM ტეგების სრული სახელმძღვანელო ყველა მარკეტინგული არხისთვის, სიებით, მაგალითებით, შემოწმების სიითა და შინაარსით»

შემდეგ

«UTM: სრული სახელმძღვანელო» — «GA4, შაბლონები Google/Meta/VK-სთვის, გავრცელებული შეცდომები». სათაური ≤ 70, აღწერილობა ≤ 120–140.

4) YouTube: «ადრე/შემდეგ»

ადრე

«როგორ ავაშენეთ SEO 30 დღეში: ყველა ნაბიჯის სრული დაყოფა აუდიტიდან განხორციელებამდე, ანალიზამდე და ანგარიშვალდებულებამდე [2025]» (105 სიმბოლო)

შემდეგ

«SEO 30 დღეში: ნაბიჯ-ნაბიჯ გეგმა» — «აუდიტი, განხორციელება, ანგარიშვალდებულება. შეცდომები და life hack-ები (2025)». სათაური ≤ 60, აღწერილობის პირველი 150–200 სიმბოლო — ძირითადი სარგებელი და CTA.

5) TikTok: «ადრე/შემდეგ»

ადრე

«ნახეთ ჩვენი ახალი გრძელი ახსნა უამრავ დეტალთან და ბმულთან ერთად — აუცილებლად წაიკითხეთ ბოლომდე!»

შემდეგ

«UTM-ში 3 შეცდომა, რომელიც ჭამს ტრაფიკს» — «შეამოწმეთ source/medium, კოდირება და გადამისამართება. სრული სახელმძღვანელო — ბმულის მეშვეობით».

6) რჩევები წინასწარი გადახედვისა და მეტამონაცემებისთვის

  • VK/Telegram-ში სწორი წინასწარი გადახედვებისთვის დააყენეთ OG მეტამონაცემები (title/description/image).
  • უნივერსალური ხილვადობისთვის შეინარჩუნეთ title ≤ 60–70, description ≤ 140–160.
  • აღწერილობის პირველ 1–2 სტრიქონში მოათავსეთ ძირითადი სარგებელი.
  • გამოქვეყნებამდე შეამოწმეთ წინასწარი გადახედვები (ექსპერტიზების/წინასწარი გადახედვის ბოტების/გაზიარების მეშვეობით).
