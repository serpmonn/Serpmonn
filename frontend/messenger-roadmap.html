Этапы разработки

Serpmonn Messenger строится поэтапно — от Go-ядра до мультиплатформенного приложения с офлайн-mesh.

  1. 01JS-прототип✓ Готово
    • Референсная реализация на Node.js + libp2p
    • Проверка P2P-модели и протокола
  2. 02Go-core: identity, crypto, chat, storage✓ Готово
    • Ed25519-идентификаторы, Curve25519 + AES-GCM
    • Структура чатов и сообщений
    • Локальное хранилище на SQLite
  3. 03Go-core net: QUIC + Relay v2 + DCUtR + GossipSub✓ Готово
    • QUIC-транспорт через libp2p
    • Relay v2 + DCUtR hole punching
    • GossipSub для групповых чатов
  4. 04Android: onboarding, чат, QR→ Сейчас
    • Kotlin + Jetpack Compose UI
    • Интеграция Go-core через gomobile
    • Onboarding, 1:1 и групповые чаты, QR-код контакта

    Текущая пре-альфа: app-debug.apk — сборка подписана debug-ключом.

  5. 05Backup, restore, linked desktopДалее
    • Зашифрованные резервные копии ключей
    • Привязка десктоп-клиента через QR
  6. 06BLE / Wi-Fi Direct meshДалее
    • Bluetooth Low Energy mesh без интернета
    • Wi-Fi Direct для локальной сети
  7. 07iOSДалее
    • Портирование на Swift/SwiftUI с тем же Go-core
