Como apoiar o projeto?

  1. Escolha um método conveniente das opções abaixo
  2. Clique no botão para doação

Todos os pagamentos são processados por meio de sistemas de pagamento seguros dos serviços selecionados.

Assinatura / Pagamento único

Boosty

  • Clique no botão à direita para ir ao Boosty
  • Clique no botão de doação (donate)
  • Disponível: SBP, cartões bancários, PayPal
  • Pagamentos mensais podem ser configurados
Ir para o Boosty
Pagamento único

DonationAlerts

  • Vá para a página de doação através do botão à direita
  • Digite o valor e clique "doar"
  • Suporta: cartões, SBP, Qiwi
  • Você pode deixar um comentário com sua doação
DonationAlerts
Transferência direta

Cartão Tinkoff / SBP

  • Transferência direta rápida para cartão Tinkoff
  • A página do banco abrirá e exigirá autorização
  • Autorize, digite o valor e confirme a transferência
  • Disponível para clientes de qualquer banco via SBP
Transferir
Carteira / Cartões

YooMoney

  • Doação por meio de arrecadação YooMoney (antigo Yandex.Money)
  • Vá para a página através do botão à direita, uma página com o botão "Doar" abrirá
  • Clique no botão "Doar". Suporta: carteira YooMoney, cartões bancários, SBP
  • Transferência anônima é possível
YooMoney
Via rede social

Doação única no VK

  • Vá para a comunidade VK através do botão à direita
  • Encontre qualquer post com o botão "Enviar doação"
  • Clique no botão e selecione o valor
  • Pagamento via VK Pay (cartões, carteiras)
