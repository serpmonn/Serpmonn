Introdução

O site serpmonn.ru valoriza a privacidade do usuário e toma todas as medidas necessárias para proteger os dados pessoais. Esta política de privacidade descreve como e para quais finalidades os dados do usuário são coletados, usados, armazenados e protegidos. Ao usar este site, o usuário concorda com os termos estabelecidos nesta política.

1. Dados Coletados

O site pode coletar os seguintes tipos de dados:

  • Dados pessoais: nome, endereço de e-mail, número de telefone.
  • Dados técnicos: endereço IP, tipo de navegador, sistema operacional, informações de visita ao site (incluindo páginas visitadas, data e hora da visita).
  • Dados fornecidos pelo usuário: informações fornecidas pelo usuário ao preencher formulários no site, incluindo comentários, feedback, solicitações e outros feedbacks.

2. Uso de Dados

Os dados são usados para os seguintes propósitos:

  • Fornecer e melhorar os serviços.
  • Processar solicitações e feedback.
  • Para análise e pesquisa para melhorar o desempenho do site e garantir a segurança.
  • Proteger contra acesso não autorizado e prevenir ataques ao site.
  • Cumprir obrigações com os usuários.

3. Armazenamento de Dados

Os dados são armazenados pelo tempo necessário para atingir os propósitos especificados:

  • Dados pessoais (nome, e-mail, telefone) são armazenados até que sua exclusão seja solicitada ou os serviços sejam encerrados.
  • Dados técnicos (endereços IP, dados do navegador) são armazenados por não mais de 30 dias.

4. Compartilhamento de Dados com Terceiros

Os dados pessoais dos usuários não são compartilhados com terceiros.

5. Proteção de Dados

Todas as medidas necessárias são tomadas para proteger os dados contra acesso, uso ou divulgação não autorizados. Os dados pessoais são protegidos usando tecnologias modernas de criptografia e outros métodos de segurança. No entanto, apesar das medidas tomadas, a segurança absoluta não pode ser garantida.

6. Direitos do Usuário

O usuário tem o direito de:

  • Solicitar acesso aos dados pessoais armazenados.
  • Solicitar correção ou exclusão de dados.
  • Opor-se ao processamento de dados em determinados casos.
  • Retirar o consentimento para o processamento de dados pessoais.

7. Informações de Contato

Para perguntas relacionadas à política de privacidade, processamento de dados ou exercício de direitos, entre em contato: privacy@serpmonn.ru.

Termos de serviço e aviso legal estão disponíveis na seção de Aviso Legal.

8. Alterações na Política de Privacidade

Todas as alterações na política de privacidade são publicadas nesta página. É recomendado verificar regularmente se há informações atualizadas.
