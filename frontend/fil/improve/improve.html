Ano ang Pagbutihin? - Mga Mungkahi at Ideya

Mahalaga ang feedback ng mga user para sa pag-unlad ng proyekto. Sabihin sa amin kung ano ang maaaring gawing mas mabuti, anong mga feature ang idagdag o kung ano ang ayusin. Ang bawat mungkahi ay isasaalang-alang.

Ang mga mungkahi ay sinusuri sa loob ng 3-5 araw ng trabaho. Ang mga kawili-wiling ideya ay idinaragdag sa plano sa pag-unlad.

Mga Estadistika ng Mungkahi

— Tinanggap na mga mungkahi
— Sa pag-unlad
— Naipatupad na
— Sa linggong ito

Anong mga mungkahi ang tinatanggap?

🎯 Pag-andar

  • Mga bagong feature
  • Pagpapabuti ng umiiral
  • Mga pagsasama sa mga serbisyo

🎨 Disenyo

  • Interface at UX
  • Mga scheme ng kulay
  • Responsiveness

⚡ Pagganap

  • Bilis ng trabaho
  • Pag-optimize
  • Katatagan

📱 Nilalaman

  • Mga bagong seksyon
  • Mga format ng materyales
  • Kapaki-pakinabang na mga mapagkukunan

Mga Halimbawa ng Mungkahi

Magdagdag ng kasaysayan ng pagbabago

"Iminumungkahi kong magdagdag ng isang pahina na may kasaysayan ng update upang makita kung anong mga pagbabago ang ginagawa sa proyekto"

I-export ang data sa PDF

"Magiging maginhawang magdagdag ng pindutan ng export para sa mga resulta sa PDF format para sa pag-print"

Mga abiso tungkol sa mga bagong artikulo

"Gusto kong makatanggap ng mga newsletter sa email tungkol sa mga bagong materyales minsan sa isang linggo"

Magsumite ng mungkahi

Proseso ng pagsusuri ng mungkahi

1. Moderation — pagsusuri ng mungkahi sa loob ng 1-2 araw

2. Pagtatasa — pagsusuri ng pagiging kumplikado, kapakinabangan at pagkaapurahan

3. Pagpaplano — pagdaragdag sa development backlog

4. Pagpapatupad — ang pinakamahusay na mga mungkahi ay ipinatutupad sa mga darating na update

5. Feedback — kung ibinigay ang email, ipaalam ang status ng ideya
