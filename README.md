## Обзор

Serpmonn - это удобный набор инструментов и развлечений с поиском без рекламы и слежки. Всё в одном месте: мультимодальный AI-поиск, новости, игры, инструменты и промокоды!


## 🚀 Ключевые возможности


- **🔐 Безопасная аутентификация** - PASETO токены, bcrypt хеширование, восстановление пароля
- **🤖 Telegram-боты** - Игры, мемы, подтверждения,
- **📰 Новостной блок** - Агрегация через SearXNG + Ollama, 12 категорий, hero-карточка + лента, поддержка 40+ языков
- **🎮 Игровая платформа** - Интерактивные игры с таблицами лидеров
- **📱 PWA поддержка** - Прогрессивное веб-приложение
- **📧 Email сервисы** - Автоматическая отправка писем, OnnMail система
- **🔧 Автоматизация** - счётчики, подписчики
- **👤 Профили пользователей** - Управление профилями и настройками
- **🔖 Находки (Findings)** - Сохранение ответов ИИ, публичная лента, сообщения между пользователями, подписки, лайки, комментарии и уведомления
- **🔍 Поисковая система** - Интегрированный поиск по контенту
- **📊 Аналитика** - Счетчики и метрики

## 🔖 Находки (Findings)


Социальный слой поверх AI-поиска:

1. **Сохранение** — пользователь сохраняет ответ ИИ (snapshot: текст, источники, медиа) в профиль.
2. **Видимость** — private, public (лента), link, followers; отправка другу через inbox.
3. **Лента** — публичные находки, лайки, комментарии, просмотры, подписка на автора, «сохранить себе».
4. **Сообщения** — прямой обмен находками между пользователями.
5. **Уведомления** — новые публикации подписок и комментарии.

Frontend: `findings-client.js`, `findings-modals.js`, `finding-list-card.js`. Backend: `backend/findings/`.

## Как это работает

- **Сквозное шифрование** — сообщения шифруются на устройстве и расшифровываются только у получателя. Никто — ни серверы, ни Serpmonn — не имеет доступа к содержимому.
- **Прямое P2P-соединение** — устройства общаются напрямую, без сервера переписки.
- **Резервный канал** — при жёстких сетевых ограничениях трафик идёт через посредника, оставаясь E2E-зашифрованным.
- **Мобильное приложение** — нативный Android-клиент: личные и групповые чаты, обмен контактами по QR, локальная история на устройстве.

## Архитектура (кратко)
