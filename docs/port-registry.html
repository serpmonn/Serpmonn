Port Registry

Карта портов сервера

Все занятые порты, их назначение и диапазоны. Обновлять при добавлении нового сервиса.

Принцип распределения: Каждому типу сервиса отведён свой диапазон — это упрощает навигацию, дебаг и onboarding. При добавлении нового сервиса выбирать порт из соответствующего диапазона. Порт 8003 — временный (VS Code Remote SSH), освобождается после закрытия сессии IDE.
