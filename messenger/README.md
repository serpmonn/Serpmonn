Децентрализованный мессенджер с end-to-end шифрованием
Сообщения передаются напрямую между устройствами.
Никто — ни серверы, ни разработчики — не имеет доступа к вашей переписке.

Требуется Android 8.0+. Личные и групповые чаты доступны в тестовой сборке.

Как это работает
Сквозное шифрование — сообщения шифруются на устройстве и расшифровываются только у получателя.
Прямое соединение — устройства общаются напрямую, без сервера переписки.
Резервный маршрут — при жёстких сетевых ограничениях трафик идёт через посредника, оставаясь зашифрованным.

Установка APK
На Android откройте Настройки → Безопасность и разрешите установку из неизвестных источников.
Скачайте универсальный APK со страницы мессенджера.
Откройте загруженный файл и установите.
Сборка подписана debug-ключом — предупреждение Android ожидаемо для пре-альфы.
