# RuStore: сдача Serpmonn (чеклист модерации)

Package: `ru.serpmonn` · versionName `1.2` · versionCode `3`

Официально: [требования к приложениям](https://www.rustore.ru/help/developers/publishing-and-verifying-apps/requirement-apps), [WebView](https://www.rustore.ru/help/users/start/installation-promotion-and-publication).

## 1. Что уже в APK (для модератора)

- Нативная оболочка Capacitor (не Chrome / не TWA, без адресной строки)
- Портрет
- Свой splash / иконка Sn
- Экран «нет сети» с подсказками и автоповтором при появлении сети
- RuStore Push SDK (`pushclient`) + сервис пушей + запрос `POST_NOTIFICATIONS`
- Контент с `serpmonn.ru` внутри WebView (`?app=1` — без cookie-баннера сайта)

## 2. Перед загрузкой сборки

1. В [RuStore Консоль](https://console.rustore.ru/) → приложение → **Push-уведомления → Проекты**:
   - создай проект (package `ru.serpmonn`, отпечаток той же подписи, что у APK);
   - скопируй **ID проекта** в `android-app/rustore.properties`:
     ```
     pushProjectId=ВАШ_ID
     ```
2. `keystore.properties` заполнен (release).
3. Собери release: `npm run build:release`.
4. На устройстве с установленным **RuStore** проверь: запуск, offline (авиарежим), разрешение уведомлений.

## 3. Карточка приложения (тексты)

**Краткое описание (до ~80 символов):**  
Агентный поиск, новости, инструменты и игры Серпмонн — в одном приложении.

**Полное описание (черновик):**  
Serpmonn (Серпмонн) — приложение для умного поиска, чтения новостей и базы знаний, набора веб‑инструментов и своих игр.  
Работает онлайн: ответы ИИ и актуальный контент приходят с серверов Серпмонн. Без интернета приложение показывает понятный экран «нет сети» и позволяет повторить подключение.  
Поддерживаются push‑уведомления через RuStore.  
Официальный сайт: https://serpmonn.ru

**Категория:** Инструменты / Продuctive (уточнить в консоли)  
**Возраст:** 12+ или по фактическому контенту новостей  
**Язык:** русский
